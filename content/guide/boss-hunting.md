## 1. 보스 헌팅 개요

이 모드팩에서 보스 사냥은 **장비 진행의 핵심 루프**입니다. 일반 던전·제작만으로는 도달할 수 없는 최상위 무기·방어구·소재가 보스 드롭으로만 나옵니다.

**보스를 잡는 이유:**
- **무기/방어구**: Cataclysm 보스 무기(영혼 분쇄자·아스트라페·견착식 위더 무기 등), Ice and Fire 드래곤스틸 장비, Aether 발키리·피닉스 세트 등은 **해당 보스 처치 소재**로만 얻습니다. → 무기 상세 `../위키/5_무기_위키.md`
- **진행 잠금 해제**: 황혼의 숲은 **보스를 순서대로 잡아야** 다음 구역 마법 장벽이 풀립니다(강제 진행). 보스 처치 = 다음 콘텐츠 열쇠.
- **소재·재료**: 드래곤스틸 주괴(성체 드래곤), 잊혀진 주괴, 워든 심장 등 가공·강화 재료.
- **음반·트로피**: Cataclysm 보스는 고유 음반을, 황혼의 숲 보스는 트로피를 드롭(받침대 장식용).

> 보스 ↔ 구조물 매핑 전체는 `../위키/9_구조물_위키.md` 4·5장을 함께 보세요.

## 2. 사전 준비 체크리스트

보스마다 위협 유형이 다릅니다. **공통 준비물 + 상황별 추가 준비**로 나눠 챙기세요.

### 공통 (모든 보스전)
- [ ] **방어구**: 현 단계 최고 등급 풀세트 + Apotheosis 접사/젬 강화 (등급·접사가 생존을 좌우)
- [ ] **무기**: 주력 근접 + 원거리(보스가 공중/수중일 때 대비)
- [ ] **회복**: 즉시 회복 물약 다수, 재생 물약, 황금 사과/마법이 깃든 황금 사과
- [ ] **음식**: 포화도 높은 음식 충분히 (전투 장기화 대비)
- [ ] **버프 물약**: 힘, 신속, 저항, 흡수(생명) — 양조해 미리 준비
- [ ] **토템(언데드의 토템)**: 한 방 사고 보험. 보조손에 들기
- [ ] **거점**: 후술 일반 팁 참고 — 흑요석/방어 블록, 사다리·물통(낙하·도주용)

### 상황별 추가
| 보스 유형 | 추가 준비물 |
|---|---|
| **화염 계열** (이그니스 / 임모레이터 / 파이어 드래곤 / Ur-Ghast / Obsidilith) | **화염 저항 물약**(장시간), 화염저항 인챈트, 네더 벽돌/흑요석 거점 |
| **얼음·냉기 계열** (말레딕투스 / Night Lich / 눈의 여왕 / 아이스 드래곤 / Frostmaw) | 빙결 대비 이동기, 알파 예티 모피(빙하 추위 저항 — TF 진행), 따뜻한 음식 |
| **수중 보스** (레비아탄 / 씨 서펀트) | **수중 호흡** + **물갈퀴(심해 가속)** 인챈트, 돌고래의 은총, 삼지창류 |
| **공중 보스** (스킬라 / 드래곤 / Ur-Ghast / 엠피데어) | **원거리 무기·낙하 보호** 인챈트, 페가수스/와이번 등 비행 탈것 또는 엘리트라 |
| **독·석화** (고르곤 / Void Blossom / 미르멕스) | 우유(독 해제), 고르곤은 **시선 회피**(석화 방지) 필수 |
| **공허·텔레포트** (엔더 수호자 / Obsidilith / Night Lich) | 엔더 진주, 텔레포트 추적 대비 기동 |
| **폭발·자폭** (하빈저 — "자폭 시퀀스") | 저항·흡수 버프, 폭발 보호 인챈트, 충분한 거리 |

> 💡 모드팩에 **소환수·길들임 보조 몹**이 있다면 미리 데려가 탱킹/딜에 활용(아기 레비아탄·현세의 잔재 등은 길들여 아군화 가능 — 4-2 참고).

## 4. 주요 보스별 공략 카드

각 카드: **위치/도달 → 소환법 → 위험 패턴 → 핵심 전리품**. 패턴은 파일 근거 또는 일반적 설명만 기재합니다.

### 4-1. Twilight Forest
**한국어 번역 있음 ✅ — 진행 순서 강제(`tfEnforcedProgression`).** 특정 보스를 잡기 전엔 다음 구역 바이옴이 **마법 장벽으로 잠겨** 있습니다. 어드밴스먼트에서 진행 상황 확인 가능.

> **진행 사슬(파일 근거):**
> 나가 → 리치 → 미노쉬룸(불 늪지 열기 적응) → … → 우두머리 예티(빙하 추위 저항) → Ur-Ghast → 눈의 여왕 → 고원(최종 성은 **미완성/WIP**, "진행의 끝").

#### 🐍 나가 (Naga) — 1단계
- **위치**: 나가의 정원(Naga Courtyard)
- **소환/도달**: 구조물에 자연 존재. 입장 즉시 전투.
- **위험 패턴**: 빠른 돌진·몸통 휘두르기(파일에 수치 없음 — 확인 필요). 기둥 뒤로 시야를 끊으며 회피.
- **핵심 전리품**: **나가 비늘**(Naga Scale) → 흉갑·레깅스 제작(어드밴스먼트 "네 비늘 쩔더라"). 나가 트로피.
- **잠금 해제**: 처치 시 **리치 성을 둘러싼 장벽 마법 해제**(어드밴스먼트 "나가 사냥꾼").

#### 💀 리치 (Lich) — 2단계
- **위치**: 리치 타워(Lich Tower) 꼭대기
- **위험 패턴**: 3페이즈. 1페이즈 마법 반사(쉴드) → 화살로 반사시켜 깸, 2페이즈 리치 미니언 소환, 3페이즈 근접. 리치 볼트/폭발성 볼트 주의.
- **준비물**: 활 다수(쉴드 반사용), 미니언 처리용 광역.
- **핵심 전리품**: **힘의 셉터(Scepter)** 류 — 4종 모두 모으는 도전 과제 존재("우리의 힘을 합친다면!"). 리치 트로피.

#### 🍄 미노쉬룸 (Minoshroom) — 3단계
- **위치**: 미노타우르스 미로(Labyrinth) 지하
- **위험 패턴**: 돌진·대지 강타.
- **핵심 전리품**: 미노쉬룸 트로피. **미노타우르스 스트로가노프**(섭취 시 **불 늪지의 열기에 적응** — 다음 구역 진입 조건, 어드밴스먼트 "강력한 스트로가노프!").

#### 🐉 히드라 (Hydra)
- **위치**: 히드라의 굴(Hydra Lair)
- **위험 패턴**: 다두(多頭) 보스. 화염 브레스 + **히드라 박격포**(곡사 폭발). 각 머리 개별 처치.
- **준비물**: 화염 저항, 넓은 회피 공간.
- **핵심 전리품**: 히드라 트로피, 히드라 키친(소재).

#### ⚔️ 유령 기사 (Knight Phantom)
- **위치**: 기사 요새(Knight Stronghold)
- **위험 패턴**: 다수의 유령 기사가 부유하며 검·창 투척.
- **잠금/보상**: 처치하면 **카미나이트 탑 장치가 복종**(어드밴스먼트 "카미나이트 적응"). 기사석(Knightmetal) 장비 소재.

#### 그 외 후반 (Ur-Ghast / 우두머리 예티 / 눈의 여왕)
- **Ur-Ghast(유어 가스트)**: 어둠의 탑(Dark Tower). 공중 화염 보스. 보상으로 **불타는 붉은 눈물**(어드밴스먼트 "불의 눈물").
- **우두머리 예티(Alpha Yeti)**: 예티 동굴. 처치 시 **알파 모피**로 옷을 지어 **빙하 추위 저항**(어드밴스먼트 "알파의 모피") → 오로라 궁전 진입 가능.
- **눈의 여왕(Snow Queen)**: 오로라 궁전 꼭대기. 처치 시 "맑은 하늘"(빙하 정복). 눈의 여왕 트로피.

> 모든 트로피는 **어둠의 숲 폐허의 트로피 받침대**에 장식(어드밴스먼트 "트로피 챔피언").

### 4-2. Cataclysm
**한국어 번역 있음 ✅ — 모드팩 최상위 보스군.** 각 보스는 **고유 구조물 + 소환 의식**으로 깨웁니다. 전 보스 처치 시 "대격변을 건넌 자".

#### 🔥 이그니스 (Ignis)
- **위치**: **불타는 투기장(Burning Arena)** — 네더. 진입 시 "점화된 광전사" 등장.
- **소환**: **타오르는 잿더미를 (불의) 제단에 바쳐** 소환(어드밴스먼트 "꺼져버린 불꽃"). 처치 시 "네더의 망자들이 깨어난다".
- **위험 패턴**: 화염구(이그니스의 화염구 / **심연 화염구**) 투사.
- **준비물**: **화염 저항 필수**(장시간), 화염저항 인챈트.
- **핵심 전리품**: **소각자(The Incinerator)**, 임모레이터/타오르는 손길 계열, 이그니스 음반("Light My Fire"). → 무기 위키 3-5장.

#### 🌑 네더라이트 괴물 (Netherite Monstrosity)
- **위치**: **고대 공장(Ancient Factory)** — "어울리지 않는" 산업 구조물.
- **소환**: 구조물 내 자연 존재(처치 어드밴스먼트 "망가진 네더의 전쟁병기"). 별도로 **네더라이트 조각상(Netherite Effigy)** 으로 **네더라이트 미니스트로시티**(소형 버전)를 소환·용암 전지로 길들임 가능.
- **위험 패턴**: 거대 근접 강타·범위 충격파(수치 미기재 — 확인 필요).
- **핵심 전리품**: **절멸자(The Annihilator)**, 견착식 위더 무기(Wither Assault Shoulder Weapon — 위더 미사일/곡사포), 네더라이트 괴물 음반("Spawn Of Hell").

#### 🌊 레비아탄 (The Leviathan)
- **위치**: **가라앉은 도시(Sunken City)** — 수중.
- **소환**: **심연의 제물(Abyssal Sacrifice)을 (심연의) 제단에 바쳐** 소환(어드밴스먼트 "다차원의 포식자").
- **⚠️ 핵심 주의**: **레비아탄은 물 밖에선 무적**(파일 명시: "레비아탄은 물 밖에선 무적입니다!"). 반드시 **수중에서 교전**해야 데미지가 들어감.
- **준비물**: 수중 호흡 + 물갈퀴(심해 가속) 인챈트, 돌고래의 은총, 삼지창. 처치 시 산호 골렘 출현(구조물 위키).
- **핵심 전리품**: **아스트라페(Astrape — 번개 창)** / **케라우노스(Ceraunus — 닻 투척, 웅크려 파도)**, 타이달 클로, 레비아탄 음반("Predator of The Abyss"). **아기 레비아탄**을 양동이로 포획·사육 가능.

#### 🏜️ 고대의 잔재 (Ancient Remnant)
- **위치**: **저주받은 피라미드(Cursed Pyramid)** — 사막.
- **소환**: 피라미드 내부 **수상한 모래(Suspicious Sand)를 발굴**해 각성시킨 뒤 처치(어드밴스먼트 "죽음은 스스로 죽을 길을 찾는다는 거죠").
- **위험 패턴**: 모래폭풍·매장형 공격(일반 설명 — 수치 확인 필요).
- **핵심 전리품**: **사막의 분노(저주받은 모래폭풍 3개)**, **고대의 창(공격 시 모래폭풍)**, 코피스/의식용 단검, 음반("The Dryest Beast"). 별개로 **현세의 잔재(Modern Remnant)** 는 **잔재의 두개골**로 소환(스니퍼 알로 길들임) → 미트 슈레더/레이저 개틀링 계열.

#### ❄️ 말레딕투스 (Maledictus)
- **위치**: 서리 감옥(Frosted Prison) 추정 구조물.
- **소환**: **묘비(Tombstone)를 눌러** 소환(어드밴스먼트 "저주에 맞서는 성전").
- **준비물**: 냉기 대비.
- **핵심 전리품**: 말레딕투스 음반("Maledictus"), 사막·저주 계열 무기 소재.

#### ⚡ 스킬라 (Scylla, 폭풍의 여제)

- **소환**: 구조물 내 처치(어드밴스먼트 "폭풍 후의 고요함").
- **준비물**: 공중전 — 비행 탈것/엘리트라 또는 원거리 무기.
- **핵심 전리품**: 케라우노스(Scylla Ceraunus 연계), 스킬라 음반("Scylla").

#### 👁️ 엔더 수호자 (Ender Guardian) / 엔더 골렘

- **소환**: 구조물 내 처치(어드밴스먼트 "수호자의 비밀"). 엔더 골렘은 "팔레트 스왑".
- **위험 패턴**: **엔더 수호자 탄환** 투사, 텔레포트.

### 4-3. Bosses of Mass Destruction
**미번역(영문 + 의역).** 대부분 **고정 아레나보다 차원 조건·소환 블록형**. 각 보스 처치 시 차원 강화 소재 드롭(설정에서 사망 시 고대 잔해/기둥 생성 옵션 확인됨).

#### 🌋 네더 건틀릿 (Nether Gauntlet)
- **위치**: 네더. 관련 블록 `Ancient Carved Blackstone`(고대 조각 흑암석).
- **위험 패턴(자막 근거)**: **회전 펀치(Spin Punch)**, **레이저 충전(Laser Charge)** 후 발사, 시전(Cast).
- **준비물**: 화염 저항, 레이저 충전 타이밍에 엄폐.
- **핵심 전리품**: 설정에 **사망 시 고대 잔해(Ancient Debris) 생성** 옵션 — 네더라이트 보급원. 어드밴스먼트 "Through the Gauntlet".

#### 🪨 옵시딜리스 (Obsidilith)
- **위치**: 엔드. 소환 블록 **Obsidian Altar(흑요석 제단)** + Obsidian Rune.
- **소환**: 흑요석 제단에 룬을 배치해 활성화(구조물형).
- **위험 패턴(자막 근거)**: **에너지 쉴드**, 충전 후 **에너지 분출(Burst)**, **텔레포트**, **불 분출(파동 Wave)**.
- **준비물**: 텔레포트 추적·화염 파동 회피 공간.
- **핵심 전리품**: 설정에 **사망 시 기둥(Pillar) 생성** 옵션. 어드밴스먼트 "Obsidian Obliterator".

#### 🌙 밤의 리치 (Night Lich)
- **출현**: 오버월드 밤. **소울 스타(Soul Star)** 로 소환 — 소울 스타는 **특정 몹을 일정 수 처치하면 드롭**(설정 "Kills per soul star"). 소울 스타를 떨어뜨려 소환.
- **위험 패턴(자막 근거)**: **얼음 구슬 다발(Iceballs)** 발사, **혜성(Comet)** 발사, **격노(Rage)**, **텔레포트**.
- **준비물**: 냉기 대비, 원거리 회피.
- **핵심 전리품**: 어드밴스먼트 "Night Vanquisher". (정확한 드롭 확인 필요)

#### 🌸 공허의 꽃 (Void Blossom)
- **출현**: **Void Blossom Boss Spawner** 블록으로 소환(식물형 보스).
- **위험 패턴(자막 근거)**: **굴 파기(Burrow)** 후 등장, **가시(Spike)** 돌출, 낙하 공격(Fall).
- **드롭**: **크리스탈 과일(Crystal Fruit)** — 잠시 재생+저항 부여(툴팁). 어드밴스먼트 "Extreme Weeding".

## 5. 보스전 일반 팁

- **거점 구축**: 아레나 외곽에 흑요석/방어 블록으로 **엄폐물·후퇴 지점**을 만든다. 화염 보스는 네더 벽돌/흑요석, 폭발 보스는 두꺼운 벽.
- **도주로 확보**: 사다리·물통·엔더 진주를 항상 휴대. 체력이 위험하면 즉시 빠져 회복 후 재진입.
- **소환수·아군 활용**: 길들인 보조 몹(아기 레비아탄·현세의 잔재·네더라이트 미니스트로시티 등)으로 어그로 분산. 마법 진영의 소환 주문도 탱킹용으로 유용.
- **탈것**: 공중 보스(드래곤·스킬라·Ur-Ghast)는 **비행 탈것(페가수스·와이번 등)** 또는 엘리트라로 고도 맞춰 교전. 지상 보스는 빠른 탈것으로 카이팅.
- **물약 타이밍**: 화염/냉기 저항은 **교전 직전** 갱신(지속시간 관리). 전투 중에는 즉시 회복 + 흡수 위주.
- **무적 프레임 이해**: 일부 보스 무기(미트 슈레더)는 무적 프레임을 무시 — 다단히트 무기로 보스를 끊지 않고 갉아낼 수 있음.
- **진행 순서 준수(TF)**: 황혼의 숲은 장벽이 잠겨 있으니 **반드시 나가→리치→… 순서**대로. 어드밴스먼트 화면으로 다음 목표 확인.
- **물 밖 무적 보스(레비아탄)**: 절대 물 밖으로 끌어내지 말 것 — 무적이 됨. 반대로 **약점 환경(수중/시선 회피/외눈)** 을 강제하는 보스는 그 조건을 먼저 맞춘다.
