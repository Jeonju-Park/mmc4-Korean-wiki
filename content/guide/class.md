# ⚔️ 직업(스킬트리) 가이드 — Medieval MC MMC4

> Medieval MC MMC4 (1.20.1 Forge)의 **Skill Tree(패시브 스킬트리)** 모드 직업 시스템 총정리.
> 6직업 — **연금술사 · 대장장이 · 요리사 · 마법부여사 · 사냥꾼 · 광부**.
> 모든 노드 이름은 모드팩 실제 데이터(`skilltree` 모드 언어 파일)에서 추출했습니다.
> ⚠️ 노드별 **수치(%·증가량)는 게임 내 버전·설정에 따라 달라지는 데이터 구동값**입니다. 정확한 수치는 **반드시 게임 내 노드 툴팁**을 확인하세요.

---

## 1. 스킬트리 시스템 개요

종족(Origins)과는 **완전히 별개**인 시스템입니다. `O` 또는 스킬트리 단축키(`Open Skill Tree`)로 거대한 패시브 노드 맵을 엽니다.

| 요소 | 설명 |
|---|---|
| 🎯 **스킬 포인트 획득** | **경험치 레벨을 소모**해 포인트를 구매합니다. 화면의 **`스킬 포인트 구매`**(`widget.buy_skill_button`) 버튼 사용. 즉 레벨을 올릴수록 포인트가 쌓입니다. |
| 📈 **경험치 보너스** | 일부 노드/장신구는 **분당 경험치(Experience Per Minute)**를 올려 포인트 수급을 가속합니다. 경험치 원천: **몹 · 광석 · 낚시**. |
| 🧭 **전직(직업 선택)** | 트리에서 원하는 **직업 노드(`*_class`)**에 도달해 찍으면 그 직업이 됩니다. 한 캐릭터가 여러 트리를 넘나들 수 있습니다(포인트만 충분하면). |
| 🔼 **노드 등급** | 일반 노드 → **주요(Notable, `*_notable_*`)** → **키스톤(Keystone, `*_keystone_*`)**. 키스톤은 강력하지만 보통 **트레이드오프**가 있습니다. |
| 👑 **마스터리(`*_mastery`)** | 각 직업의 정체성을 정의하는 **핵심 패시브**. 그 직업을 했다면 사실상 목표로 삼는 노드. |
| 🌀 **게이트웨이(`*_gateway`)** | 멀리 떨어진 트리를 잇는 **순간 연결 통로**. 같은 이름의 관문끼리 연결됩니다 → **혼합(하이브리드) 빌드** 가능. |
| 🔱 **하위직업(Subclass)** | 각 직업은 **2개의 하위직업**으로 갈라집니다(예: 광부 → 탐험가 / 보석세공사). 추가로 트리 깊숙한 곳에 **특수 노드(`subclass_special`)**가 1개씩 있습니다. |
| 📜 **지혜의 두루마리(Wisdom Scroll)** | 사용 시 **패시브 스킬 포인트 1개**를 즉시 부여. |
| 🌫️ **망각의 두루마리(Amnesia Scroll)** | **스킬트리 전체 초기화(리스펙)**. ⚠️ 초기화 시 **스킬 포인트의 일부(%)를 잃습니다** — 신중하게. |

### 게이트웨이 연결 지도 (혼합 빌드 참고)

| 관문 이름 | 연결되는 직업들 |
|---|---|
| 🟣 **영혼의 관문(Spiritual)** | 연금술사 ↔ 요리사 |
| 🔵 **차원의 관문(Dimensional)** | 대장장이 ↔ 사냥꾼 |
| 🟡 **천상의 관문(Astral)** | 마법부여사 ↔ 광부 |

> 같은 관문을 공유하는 두 직업은 **자연스러운 하이브리드 짝**입니다. (예: 광부로 자원 모으며 마법부여사 화력을 일부 챙기기.)

## 2. 직업 티어 표 (추천도)

> 추천도는 **싱글/생존 진행 편의 + 전투 안정성 + 모드팩 시너지** 기준의 가이드 의견입니다. 실제 우열은 빌드·플레이스타일에 좌우됩니다.

| 직업 | 추천도 | 한 줄 콘셉트 | 추천 근거 |
|---|:---:|---|---|
| ⛏️ **광부(Miner)** | ⭐⭐⭐⭐⭐ | 자원·젬 수급 + 채굴 생존 | Apotheosis 젬/소켓 경제와 직결, 진행 전반을 가속. **필수급** |
| 🍳 **요리사(Cook)** | ⭐⭐⭐⭐⭐ | 잘 먹고 강해지는 근접 탱커 | 농부의 기쁨 음식과 시너지, 초반 생존이 매우 쉬움. **추천** |
| 🏹 **사냥꾼(Hunter)** | ⭐⭐⭐⭐ | 활/원거리 + 화살통 + 보물 | 안전한 원거리 딜, **보물 사냥꾼** 마스터리로 전리품 이득 |
| 🛡️ **대장장이(Blacksmith)** | ⭐⭐⭐⭐ | 장비 제작 강화 + 방패 탱커 | 직접 만든 방어구/무기/방패 강화, Tetra 장비와 궁합 |
| ⚗️ **연금술사(Alchemist)** | ⭐⭐⭐ | 물약·독 활용 근접 암살/힐러 | 강력하지만 **물약 혼합 운영**이 필요해 손이 감 |
| ✨ **마법부여사(Enchanter)** | ⭐⭐⭐ | 마법 피해·룬 캐스터 | 화력 천장은 높으나 인챈트/마법 인프라 의존도 높음 |

**필수 추천**: 처음 한다면 **광부 또는 요리사**로 시작하면 진행이 가장 매끄럽습니다.
**혼합 추천**: 같은 관문을 쓰는 짝(위 표)을 섞으면 트리 이동 손실이 적습니다.

### ⛏️ 광부 (Miner)

**콘셉트**: 채굴·자원 수집에 특화된 직업. 광석에서 더 많은 전리품과 **젬(보석)**을 뽑고, 돌·동굴 환경에서 단단하게 버팁니다. 진행의 **경제 엔진** 역할.
**게이트웨이**: 🟡 천상의 관문 (→ 마법부여사와 연결)

**핵심 노드**

| 등급 | 이름 | 비고 |
|---|---|---|
| ⭐ 마스터리 | **탐욕(Greed)** | 광부의 정체성. 채굴 전리품/수급 강화 |
| 🔶 주요 | 발굴 · 저주받은 돌 · 안전모 · 치유의 수정 · 생명의 수정 · 연마 · 빛의 수정 | 채굴 생존·유틸 |
| 🔱 키스톤 | **돌의 심장 · 귀중한 무기 · 컬리넌 · 에스트렐라 데 푸라** | 강력한 채굴/전투 특화 |

**하위직업 2갈래**

- 🧭 **탐험가(Explorer)** — 동굴 탐사·이동·생존 중심. 특수 노드 **장식용 장화(Decorative Boots)**.
- 💎 **보석세공사(Jeweler)** — **젬/보석 가공·소켓** 경제 특화. Apotheosis 젬 빌드와 최고 궁합.

**추천 플레이스타일**: 초반부터 광석을 쓸어 담아 **장비 + 젬 + 경험치**를 동시에 수급. 보석세공사로 가면 좋은 젬을 모아 장비 소켓을 채우는 "꾸미기" 빌드가 강력합니다.

**🧰 만들어야 할 생산성 툴·아이템**
- **여행자의 배낭(travelersbackpack)** + **자동수집/화로 업그레이드** — 채굴 중 광석 자동 회수·제련.
- **전송석(waystones)** — 광산 거점 ↔ 본진 순간이동, 왕복 시간 제거.
- **Apotheosis 젬 + 정동석/젬 가루** — 광석에서 나오는 **젬을 모루로 부숴 젬 가루** 제작 → 장비 소켓에 삽입(보석세공사 시너지).
- **톰의 보관(toms_storage)** 터미널 — 쏟아지는 광물·젬을 통합 관리.
- **Tetra 모듈식 곡괭이** — 채굴 효율 모듈을 직접 조립.

**📜 해야 할 일 · 관련 퀘스트**
- **Welcome / Storage 챕터** — 배낭·보관 네트워크부터 갖춰 채굴 지속력 확보.
- **Oddities 챕터** — 웨이스톤, 광맥 도구 등 편의 도구.
- 광석 사냥 → 젬 수급 루프를 돌리며 **스킬 포인트(경험치)**도 함께 적립.

### 🍳 요리사 (Cook)

**콘셉트**: **잘 먹어서 강해지는 근접 탱커/브루저**. 직접 만든 음식이 회복·포만·피해를 강화하고, **배가 부를수록 전투력이 오릅니다**.
**게이트웨이**: 🟣 영혼의 관문 (→ 연금술사와 연결)

**핵심 노드**

| 등급 | 이름 | 비고 |
|---|---|---|
| ⭐ 마스터리 | **푸짐한 한 상(Large Servings)** | 요리사 정체성 — 제작 음식 효과 강화 |
| 🔶 주요 | 암브로시아 과일 · 매운 식사 · 두꺼운 팔 · 간식 · 건강한 식단 · 강타 · 에너지 비축 | 음식 버프·생존·근접 |
| 🔱 키스톤 | **과체중 · 비대한 몸 · 채식주의 · 뜨거운 음식** | 배부름·음식 기반 트레이드오프형 |

**하위직업 2갈래**

- 🪓 **광전사(Berserker)** — 포만 상태에서 폭발적 근접 화력. 특수 노드 **유해 연구(Studying Remains)**.
- 🎣 **어부(Fisherman)** — 낚시·수중 연계 보너스. (낚시 시스템은 `../상세/낚시_가이드.md` 참고)

**추천 플레이스타일**: **농부의 기쁨(farmersdelight)** 조리냄비로 좋은 음식을 대량 생산해 **항상 배를 가득 채우고**, 묵직하게 근접전. 초반 생존이 가장 쉬운 직업 중 하나.

**🧰 만들어야 할 생산성 툴·아이템**
- **조리냄비(Cooking Pot, farmersdelight)** — 스튜·다회분 요리. 요리사 보너스의 핵심 공급원.
- **부엌 칼(Cutting Board)** — 재료 손질·부산물 확보.
- **여행자의 배낭 + 음식 보관/자동급식 업그레이드** — 탐험 중 허기 유지.
- **농작물 자동화(밭·물뿌리개 등)** — 재료 안정 공급.

**📜 해야 할 일 · 관련 퀘스트**
- **Welcome 챕터** — 기본 진행·배낭.
- 농부의 기쁨 작물·요리 루프 구축이 사실상 "직업 퀘스트".
- 더 깊은 빌드/노드 운용은 **`../상세/요리사_직업_가이드.md`** 참고.

### 🏹 사냥꾼 (Hunter)

**콘셉트**: **활/석궁 원거리 딜러**. 화살 효율·치명타·속사를 올리고, **화살통(Quiver)**과 전리품 보너스로 안전하게 사냥합니다.
**게이트웨이**: 🔵 차원의 관문 (→ 대장장이와 연결)

**핵심 노드**

| 등급 | 이름 | 비고 |
|---|---|---|
| ⭐ 마스터리 | **보물 사냥꾼(Treasure Hunter)** | 전리품/행운 강화 |
| 🔶 주요 | 절약 · 정밀 · 생존 의지 · 피에 굶주린 화살 · 움직임이 곧 생명 · 숙련된 궁수 · 속사 | 원거리 딜·생존 |
| 🔱 키스톤 | **저격수 · 맞춤 방어구 · 모자 장인 · 장식용 활** | 원거리 특화 |

**하위직업 2갈래**

- 🌲 **레인저(Ranger)** — 기동·생존형 궁수. 특수 노드 **피에 굶주린 칼날(Bloodthirsty Blade)**.
- 🪶 **활 장인(Fletcher)** — **화살·활 제작/화살통** 특화. 탄약 효율 극대화.

**추천 플레이스타일**: 좋은 활 + **화살통(skilltree 모드의 Quiver)**으로 탄약을 넉넉히 들고 **거리에서 처리**. 보물 사냥꾼 마스터리로 전리품 이득까지.

**🧰 만들어야 할 생산성 툴·아이템**
- **화살통(Quiver)** — 다양한 등급(뼈·강철·다이아·화염·치유·독·은밀 등). 활 장인과 직결.
- **Tetra 모듈식 활/석궁** — 사거리·연사 모듈 조립.
- **전송석** — 사냥터 ↔ 거점 이동.
- **여행자의 배낭** — 화살·전리품 보관.

**📜 해야 할 일 · 관련 퀘스트**
- **Bosses & Weapons 챕터** — 강한 활/원거리 무기 입수.
- **Welcome / Storage 챕터** — 화살·전리품 관리 인프라.

### 🛡️ 대장장이 (Blacksmith)

**콘셉트**: **장비 제작 강화 + 방패 근접 탱커**. 본인이 만든 방어구·무기·방패의 성능을 끌어올리고, 방패를 든 채 단단하게 싸웁니다.
**게이트웨이**: 🔵 차원의 관문 (→ 사냥꾼과 연결)

**핵심 노드**

| 등급 | 이름 | 비고 |
|---|---|---|
| ⭐ 마스터리 | **흑강(Black Steel)** | 제작 장비 강화의 정점 |
| 🔶 주요 | 방패 제작자 · 충격 · 철 껍질 · 은신처 · 확고한 자세 · 평형추 · 양손잡이 | 방어·근접 |
| 🔱 키스톤 | **살아있는 요새 · 거인 · 강철 금속 · 경량 합금** | 탱커/제작 특화 |

- 🔧 **장인(Artisan)** — **수리 효율·장비 제작** 특화. 내구·경량 방패 등 제작 이득.

**추천 플레이스타일**: 자급자족형. 좋은 장비를 직접 만들어 강화하고 방패 탱킹으로 안정적인 근접전. **Tetra 모듈식 장비**와 궁합이 좋습니다.

- **톰의 보관/배낭** — 제작 재료 통합 관리.

- **Welcome 챕터의 Salvaging/Reforging Table** — 어픽스 장비 분해·재구성(제작 직업의 핵심 인프라).
- **Bosses & Weapons 챕터** — 상위 장비 재료.

### ⚗️ 연금술사 (Alchemist)

**콘셉트**: **물약·독을 활용하는 근접/유틸 직업**. 직접 혼합한 물약을 강화하고, **무기에 독을 발라** 적을 약화시킨 뒤 처리합니다.
**게이트웨이**: 🟣 영혼의 관문 (→ 요리사와 연결)

> 핵심 메커니즘: **물약 혼합(Potion Mixing)** — 제작대에서 서로 다른 두 물약을 합쳐 혼합물 제작. **무기 독바르기(Weapon Poisoning)** — 근접 무기 + 해로운 물약을 합쳐 무기에 독 부여.

**핵심 노드**

| 등급 | 이름 | 비고 |
|---|---|---|
| ⭐ 마스터리 | **비밀 재료(Secret Ingredient)** | 물약 효과 강화 |
| 🔶 주요 | 실험 · 중독 · 향상된 반사신경 · 피의 물약 · 중독성 · 잔혹 · 아드레날린 | 물약·회피·근접 |
| 🔱 키스톤 | **돌연변이 · 과다 복용 · 순수 · 지속성 독소** | 물약/독 특화 |

**하위직업 2갈래**

- 🗡️ **암살자(Assassin)** — 치명타·독 기반 폭딜. 특수 노드 **가시 반지(Spiked Rings)**.
- 💚 **치유사(Healer)** — 이로운 물약·회복·생존 강화.

**추천 플레이스타일**: 회피(Evasion) 기반으로 맞지 않으며, **독 바른 무기 + 강화 물약**으로 적을 깎아냅니다. 운영이 손은 가지만 천장이 높습니다.

**🧰 만들어야 할 생산성 툴·아이템**
- **양조기(Brewing Stand) + 물약 재료 농장** — 혼합·독 물약 원료.
- **제작대(물약 혼합/독바르기용)** — 메커니즘 발동처.
- **Relics 장신구/Apotheosis 젬** — 회피·물약 지속 보강.
- **여행자의 배낭** — 물약 다량 휴대.

**📜 해야 할 일 · 관련 퀘스트**
- **Welcome 챕터** — 기본 인프라.
- 양조 인프라 구축이 사실상 직업 진행의 핵심.

### ✨ 마법부여사 (Enchanter)

**콘셉트**: **마법 피해·룬 기반 캐스터/하이브리드**. 무기·방어구 인챈트를 강화하고, **룬(Rune)** 노드로 공격·방어를 보강합니다.
**게이트웨이**: 🟡 천상의 관문 (→ 광부와 연결)

**핵심 노드**

| 등급 | 이름 | 비고 |
|---|---|---|
| ⭐ 마스터리 | **숨겨진 지식(Hidden Knowledge)** | 마법/인챈트 강화의 정점 |
| 🔶 주요 | 마력의 흐름 · 사신 · 룬 방벽 · 에너지 프리즘 · 마력으로부터의 생명력 · 룬 칼날 · 무기 활성화 | 마법 피해·룬 |
| 🔱 키스톤 | **엑스칼리버 · 이지스 · 보호의 룬 · 파괴의 룬** | 공격/방어 룬 특화 |

**하위직업 2갈래**

- 🔥 **방화범(Arsonist)** — 화염·폭발 공격형. 특수 노드 **지옥불 화살통(Infernal Quivers)** (사냥꾼식 원거리와도 연계).
- 📚 **학자(Scholar)** — 인챈트 효율·마법 유틸 특화.

**추천 플레이스타일**: 강한 인챈트 장비를 갖춘 뒤 **마법 피해/룬 노드**로 화력을 폭발시킵니다. 인챈트 인프라(인챈팅 테이블·Apotheosis) 의존도가 높습니다.

**🧰 만들어야 할 생산성 툴·아이템**
- **Apotheosis 인첸팅 테이블 + 인챈트 재료** — 고레벨 인챈트의 핵심.
- **경험치 수급원(몹 농장/광석)** — 인챈트·스킬 포인트 동시 확보.
- **Relics 마법 장신구** — 마법 빌드 보강.
- **톰의 보관** — 인챈트 책/재료 관리.

**📜 해야 할 일 · 관련 퀘스트**
- **Welcome 챕터의 Apotheosis 인챈팅 라인** — 인챈트 시스템 셋업.
- **Bosses & Weapons 챕터** — 인챈트 대상 상위 장비.

## 4. 빠른 선택 가이드

- 🟢 **처음·솔플** → **광부**(자원/젬) 또는 **요리사**(생존).
- 🎯 **안전한 딜러** → **사냥꾼**(원거리 + 화살통).
- 🛡️ **자급자족 탱커** → **대장장이**(제작 + 방패).
- 🧪 **운영형 고숙련** → **연금술사**(독·물약).
- ✨ **마법 고화력** → **마법부여사**(룬·인챈트).
- 🌀 **혼합 빌드** → 같은 관문 짝: 광부+마법부여사 / 요리사+연금술사 / 사냥꾼+대장장이.
