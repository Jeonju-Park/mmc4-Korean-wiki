## 2. 드래곤 (Ice and Fire) — 최강 탈것 ⭐

### 2-1. 종류
- **파이어 드래곤** / **아이스 드래곤** / **라이트닝 드래곤** (+ 보스급 변종 **블랙 프로스트**)

### 2-2. 알 부화 (속성별 조건 다름)
**드래곤 알**을 속성에 맞는 환경에 오래 두면 부화하고, **가장 가까운 플레이어에게 각인(길들임)** 됩니다.
| 드래곤 | 부화 조건 |
|---|---|
| 파이어 드래곤 | **불 속**에 알을 넣고 대기 |
| 아이스 드래곤 | **물 속**에 알을 넣고 대기 |
| 라이트닝 드래곤 | **빗속**에서 대기 |

### 2-3. 성장 관리
드래곤은 **단계(Stage)** 로 성장하며, 상태창에 단계·성별·배고픔이 표시됩니다.
- **드래곤 먹이** — 성장 가속
- **병든 드래곤 먹이** — 성장 억제(작게 유지하고 싶을 때)
- **크리에이티브 드래곤 먹이** — 즉시 길들이기(크리에이티브 전용)

> 충분히 성장해야 탑승·비행이 가능합니다. 어릴 때는 작고 약합니다.

### 2-4. 명령·관리 도구
| 도구 | 용도 |
|---|---|
| **드래곤 커맨드 스틱** | 드래곤에게 **앉기 / 배회 / 호위 / 홈 지정** 명령 |
| **드래곤 뼈 피리** | 멀리 달아난 드래곤을 멈춤·소환 |
| **드래곤 뿔피리** | 드래곤을 **아이템으로 수납**해 옮기기(불/얼음 변종) |

명령 상태: "제자리에 있음(sit) / 배회(stand) / 호위(escort) / 홈 설정·삭제".

### 2-5. 드래곤 갑옷 (탈것 장착, 약→강)
구리 → 철 → 금 → 다이아몬드 → 은 → **드래곤스틸(불/얼음/번개)**
> 드래곤스틸 갑옷이 최상위. 갑옷을 입혀 드래곤의 생존력을 높이세요.

### 2-6. 특수 능력
- **브레스(brath)**: 파이어=화염, 아이스=빙결, 라이트닝=번개 충격탄을 발사.
- **비행**: 성체 드래곤은 자유 비행. (안장 장착·세부 비행 조작키는 게임 내에서 확인 권장)

## 4. Alex's Mobs 탈것

- **라비아탄** — 네더의 거대 생물. **최대 4인(라이더+3명) 탑승**. 단체 이동용.
- **코끼리**(+엄니 코끼리) — **아카시아 꽃**으로 길들임. **등에 상자(수납) 탑재** + 카펫 장식 가능.
- **스트래들보드** — 네더라이트+스트래더라이트로 **제작하는 보드**. **본 서펜트가 사는 용암 위를 서핑**. 인챈트: 화염 저항(lavawax)·점프 강화(straddle_jump)·본서펜트 중립화(serpentfriend)·파괴 시 인벤토리 귀환(board_return). **스트래들라이트 안장** 필요.
- **스펙터** — 엔드에서 **영혼 심장**으로 유인 → **리드(끈)를 걸어 탑승**하는 비행 생물.

## 5. 에테르(Aether) 차원 탈것

> ⚠️ Aether 모드는 현재 한글 번역이 거의 없어 게임 내에서 영문으로 표시될 수 있습니다.

- **모아(Moa)** — **알을 인큐베이터에서 부화**(앰브로슘 토치 연료). 색상별 **공중 다단 점프** 횟수가 다름: **블루=3단(흔함) → 블랙=8단(최고·희귀)**. 새끼는 자연의 지팡이로 정지.
- **피그(Phyg)** — 날개 달린 돼지. **안장을 얹으면 탈것**이 됩니다.
- **에어버니(Aerbunny)** — 작은 부양 토끼, 탑승 시 낙하 완화.
- **에어웨일(Deep Aether)** — 거대 비행 고래. **에어웨일 안장**(전설급, 에어웨일 킹 드롭 또는 태양의 핵+안장으로 제작) 필요.

## 6. 날개(Icarus) — 탈것이 아닌 "자가 비행" 장비

생물을 타는 게 아니라 **등에 착용하는 날개**로 직접 납니다(엘리트라 강화판).
- 종류: 깃털 / 드래곤 / 기계 깃털·가죽 / 빛 날개 — 각 **16색**. 특수 날개도 존재.
- 조작: 공중제비 가능, **웅크리면 천천히 낙하**, 갑옷이 무거우면 속도↓.
- ⚠️ **배고프면 비행 불가**("빈 속으로는 날 수 없습니다!"), 일부 차원은 비행 금지일 수 있음.

> 드래곤을 키우기 전, 또는 차원 탐험용으로 **날개**가 가장 간편한 비행 수단입니다.

## 7. 종족 자가 비행 (탈것 아님, 참고)

일부 Medieval Origins 종족은 **스스로 날 수 있습니다**:
- **발키리** — 깃털 날개 자유 비행 + 치유(언데드에 약함)
- **밴시** — 비행 + 위상 이동(유령화)
- **픽시** — 비행/활강(가죽 초과 흉갑 착용 불가)
- **요정** — 공중 부양

(종족 상세는 `레버넌트_종족_가이드.md` / `예티_종족_가이드.md` 형식 참고)

## 8. 추천 진행

1. **초반 이동**: 바닐라 말 / 코끼리(수납) / 해마(수중).
2. **첫 비행**: **Icarus 날개**(제작이 가장 빠름) 또는 **히포그리프**(토끼발).
3. **중반**: **모아/피그**(에테르) 또는 히포그리프로 안정적 비행.
4. **후반 최강**: **드래곤** 알을 부화·각인 → 단계 성장 → 드래곤스틸 갑옷 장착 → 브레스+비행으로 전투/이동 모두 지배. **라비아탄**(단체 이동)·**에어웨일**(전설)도 별미.
