## 막막한 당신에게

처음 떨어지면 모드가 너무 많아 뭘 해야 할지 막막하죠. 괜찮아요!
**아래 체크박스를 위에서부터 하나씩 달성**해 보세요. 쉬운 것부터 시작해 점점 강해집니다.

- [ ] 🔰 **게임 켜자마자 종족 선택 GUI**가 뜬다 → 마음에 드는 종족 하나 고르기

- [ ] 🔍 **JEI(우측 아이템 목록)** 사용법 익히기 — 아이템 클릭 → 조합법 보기 (`U`=용도, `R`=조합)

> 💡 이 모드팩은 **퀘스트북이 곧 길잡이**입니다. 막히면 항상 퀘스트북부터 펴 보세요.

## 🟢 1단계 — 첫날부터 정착까지 (생존 기반)

> 목표: 죽지 않고, 잘 곳과 먹을 것과 첫 장비를 마련하기.

### 생존 기초
- [ ] 나무 캐서 **기본 도구(곡괭이·도끼·검)** 만들기
- [ ] 첫날 밤 전에 **임시 거처** 짓고 침대 놓기
- [ ] 음식 5개 이상 확보해 **배고픔 채우기** (배가 부르면 체력도 빨리 참)
- [ ] **철 장비** 한 벌 맞추기 (갑옷 + 검)

### 정체성 정하기
- [ ] 🧬 **종족(Medieval Origins)** 확정 — 농사·거래가 막히는 종족(예: 레버넌트)인지 확인. 나중에 바꾸려면 **종족 구슬(Orb of Origin)** 필요
- [ ] 🌳 **직업(Skill Tree)** 첫 노드 찍기 — 경험치 레벨로 스킬 포인트 구매 → 6직업(연금술사/대장장이/요리사/마법부여사/사냥꾼/광부) 중 하나 전직 → [요리사 예시](../상세/요리사_직업_가이드.md)

### 첫 편의 장비
- [ ] 🎒 **기본 여행자 가방** 제작 (퀘스트 보상으로도 줌) → [제작법](../상세/생산성툴_가이드.md#1--여행자의-배낭-travelers-backpack)
- [ ] ⛏️ 동굴 내려가 **다이아 5개** 캐기 → [광질 가이드](../상세/광질_가이드.md)
- [ ] 🐴 **첫 탈것** 구하기 (말·해마 등 초반 이동수단) → [탈것 가이드](3_탈것_가이드.md)

✅ **1단계 클리어 조건**: 안전한 집 + 철 이상 장비 + 종족/직업 결정 + 가방.

## 🟡 2단계 — 자리 잡기 (생산·보관·입문)

> 목표: 거점을 키우고, 자동화·보관을 갖추고, 마법/탈것/던전을 처음 맛보기.

### 생산 기반 (둘 중 취향껏)
- [ ] 🏰 **마인콜로니 시작** — 보급 캠프/보급선 설치 → 마을회관 → **건축가(Builder)** 배정 → 첫 일꾼 건물(광산/벌목장/농장) 짓기
- [ ] 🌾 또는 **직접 농사·요리** 노선 — 농부의 기쁨(조리용 냄비)으로 **든든한 요리** 만들기 → [요리사 가이드](../상세/요리사_직업_가이드.md)

### 보관·이동 정리
- [ ] 📦 **톰의 보관 네트워크** 구축 — 상자에 인벤토리 커넥터 + 저장 터미널 → 창고 통합 검색 → [제작법](../상세/생산성툴_가이드.md#2--톰의-간단한-보관-toms-simple-storage)
- [ ] 🌀 **전송석(Waystones)** 2개 이상 설치해 거점↔채굴장 순간이동 → [제작법](../상세/생산성툴_가이드.md#3--전송석-waystones)
- [ ] 🎒 여행자 가방에 **제작·화로 업그레이드** 부착 (이동 중 제작·제련)

### 첫 입문 (하나씩 찍먹)
- [ ] 🔮 **마법 1종 입문** — Iron's Spells(정통 주문서) 추천. 약한 일기 → 각인 테이블로 첫 주문 시전 → [마법 총정리](../상세/모드팩_플레이_가이드.md#5-마법-체계-총정리)
- [ ] 🐉 **첫 비행** — 만들기 쉬운 **Icarus 날개** 또는 **히포그리프**(토끼발 떨구기) → [탈것 가이드](3_탈것_가이드.md)
- [ ] 🎣 **Tide 낚시 입문** — 돌 낚싯대 제작 → 미끼 채우고 첫 물고기 잡기 → [낚시 가이드](../상세/낚시_가이드.md)
- [ ] ⚔️ **첫 던전 클리어** — 가까운 던전 하나 돌고 **접사 아이템(특수효과 장비)** 1개 줍기

✅ **2단계 클리어 조건**: 생산 거점 + 통합 보관 + 전송망 + 마법/비행/던전 중 2개 이상 경험.

## 🟠 3단계 — 성장 (강화·탐험·보스)

> 목표: 장비를 본격 강화하고, 다른 차원을 탐험하고, 진짜 보스에 도전.

### Apotheosis 장비 강화 (핵심!)
- [ ] 💎 던전/보스에서 모은 장비를 **재단조 작업대**로 접사 다시 굴리기
- [ ] 🔩 **소켓의 인장**으로 무기·방어구에 소켓 뚫고 **젬** 박기 (대장장이 작업대)
- [ ] ✨ **보석 절단 작업대**로 젬 품질 업그레이드 (금이 간 → … → 완벽한) → [강화 흐름 전체](../상세/모드팩_플레이_가이드.md#1-장비-강화의-핵심--apotheosis)
- [ ] 곡괭이에도 **채굴 젬 + 광맥 채굴 인챈트** 박아 채굴 효율 폭발 → [광질 가이드](../상세/광질_가이드.md)

### 차원 탐험 (퀘스트북에 챕터별로 있음)
- [ ] 🌋 **네더** 진입 → 네더라이트 확보
- [ ] 🌲 **황혼의 숲(Twilight Forest)** 탐험 → 보스/구조물 보상
- [ ] ☁️ **에테르(Aether)** 또는 **언더가든** 등 추가 차원 1곳 발 디디기
- [ ] 🔵 **블루 스카이(에버브라이트/에버던)** — Tetra 모듈형 도구 재료 채집

### 보스 도전
- [ ] 👹 **Cataclysm 보스** 1마리 처치 → 보스 무기/세트(이그니티움·커스뮴) 노리기
- [ ] 🛡️ 얻은 보스 세트로 **네더라이트 장비 업그레이드**
- [ ] 🏗️ 콜로니에 **대학교** 짓고 연구 트리 돌려 마을 자동화 심화

✅ **3단계 클리어 조건**: 소켓·젬·접사로 강화한 장비 + 차원 2곳 + Cataclysm 보스 1킬.

## 🔴 4단계 — 엔드게임 (최강을 향해)

> 목표: 드래곤, 최종 장비, 도감 완성, 건축 대작. 막판 컨텐츠 정복!

### 드래곤 & 최종 장비
- [ ] 🐲 **드래곤 길들이기** — 속성별 환경에 알을 두고 부화·각인 → 먹이로 성체까지 키우기 → [탈것 가이드](3_탈것_가이드.md#2-드래곤-ice-and-fire--최강-탈것-)
- [ ] ⚒️ **드래곤스틸 무기·방어구** 제작 (불/얼음/번개 계열) — 사실상 최종 티어
- [ ] 🐉 드래곤에 **드래곤스틸 갑옷** 입혀 전투·이동 모두 지배
- [ ] 💠 장비를 **완벽한 젬 + 고등급 접사**로 풀강

### 최강 보스 & 마법 극대화
- [ ] ☠️ **최상위 보스(Cataclysm 엔드 보스 등)** 도전
- [ ] 🔮 선택한 마법의 **주문력을 최고 등급**까지 끌어올리기 (고대의 서 / 대마법사 마도서)

### 수집 & 건축 대작
- [ ] 📖 **낚시 도감 완성** — 전설 물고기·미다스 물고기까지 (진정한 완성가) → [낚시 가이드](../상세/낚시_가이드.md#6-낚시-미니게임--상자--도감)
- [ ] 🏛️ **건축 대작** — Structurize/domum_ornamentum로 나만의 성·마을 짓기 → [건축 가이드](../상세/건축_가이드.md)
- [ ] 🤝 콜로니를 **자급자족 대도시**로 완성 (모든 핵심 일꾼 + 병원/학교/여관)

✅ **4단계 클리어 조건**: 성체 드래곤 + 드래곤스틸 풀세트 + 최종 보스 + 수집/건축 한 가지 완성. **당신은 이제 마스터입니다!**

## 🎯 컨셉별 미니 목표 — "이런 플레이라면 이걸 노려보세요"

자기 취향대로 한 분야를 깊게 파고 싶다면, 아래 묶음을 도전 과제로 삼으세요.

### 🎣 낚시러 (Fisherman)
- [ ] 낚싯대 **미다스 등급**까지 업그레이드
- [ ] 행운 세팅(행운의 미끼 + 황금 낚싯줄)으로 **상자(Crate)** 건지기
- [ ] **용암 낚시**(네더라이트 대) + **공허 낚시**(심연의 낚싯바늘) 도전
- [ ] 낚시 도감 100% → [낚시 가이드](../상세/낚시_가이드.md)

### ⛏️ 광부 (Miner)
- [ ] **광맥 채굴 + 행운** 곡괭이로 광석 대량 수확
- [ ] **움트는 지오어(GeOre)** 거점 농장 만들어 보석 무한 재생
- [ ] 광부 직업 트리로 **젬 위력 극대화** → [광질 가이드](../상세/광질_가이드.md)

### 🏛️ 건축가 (Builder)
- [ ] **건축가의 절단기**로 재료 흡수형 골조·기와 만들기
- [ ] 가구 작업대로 테마 가구 풀세트 배치
- [ ] Structurize 도구로 **대형 구조물** 1채 완공 → [건축 가이드](../상세/건축_가이드.md)

### 🍳 요리사 (Cook)
- [ ] **편안함(Comfort) 5분 음식** + **배부름(Nourishment) 5분 음식** 상비
- [ ] "배고프지 않을 때 피해" 노드로 **만피 포만 딜러** 완성
- [ ] 잔치 음식(블록형)으로 파티 보급 → [요리사 가이드](../상세/요리사_직업_가이드.md)

### ⚔️ 전사 (Warrior)
- [ ] **Cataclysm 보스 무기**(파이널 프랙탈·츠바이엔더 등) 1개 획득
- [ ] Better Combat 콤보로 던전 보스 솔로 클리어
- [ ] Apotheosis 접사·젬으로 **신화 등급 이상** 무기 완성 → [무기 티어](../상세/모드팩_플레이_가이드.md#2-무기-티어)

### 🔮 마법사 (Mage)
- [ ] Iron's Spells **학파 1종 주문력 최대**까지 육성
- [ ] 또는 Ars Nouveau로 **나만의 자작 주문** 만들기
- [ ] Eidolon(흑마법)으로 언데드 군대 부리기 → [마법 총정리](../상세/모드팩_플레이_가이드.md#5-마법-체계-총정리)

### 🏰 콜로니장 (Colony Lord)
- [ ] 시민 20명 이상 마을 운영
- [ ] 대학교 연구 트리 절반 이상 해금
- [ ] 광산·벌목장·농장·대장간으로 **완전 자급자족** 달성

### 🐉 조련사 (Tamer)
- [ ] 드래곤 3속성(불/얼음/번개) 모두 각인
- [ ] 라비아탄(4인 탑승)·에어웨일(전설) 등 희귀 탈것 수집
- [ ] 모든 탈것에 최고 티어 갑옷 장착 → [탈것 가이드](3_탈것_가이드.md)
