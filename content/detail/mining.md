# ⛏️ 광질(채굴·광물) 가이드

> Medieval MC MMC4 — 광물 종류·채굴 도구·광맥 채굴·광부 직업 총정리
> 모드 실제 데이터에 근거합니다.

---

## 1. 무엇을 캐야 하나 (우선순위 요약)

1. **Apotheosis 젬(Gem) 재료** — 곡괭이·무기·방어구·장신구에 박는 보석. 광부 직업이 젬 위력을 크게 키워줌. 곡괭이 젬은 채굴 시 추가 드롭까지 줌.
2. **GeOre 정동석(움트는 지오어)** — 거점에 두면 보석·금속이 **무한 재생**. 다이아·에메랄드·루비·사파이어·희귀 금속까지 자급.
3. **Forbidden Arcanus 비전 수정 광석 / 데오룸·스텔라라이트** — 마법·고급 도구.
4. **Ice and Fire 은·사파이어 광석** — 은 장비(드래곤 특효), 드래곤 제작.
5. **Deeper and Darker 음울암 광석 / 레조나리움** — 후반 동굴 콘텐츠.

> ⚡ **광맥 채굴(Vein Mining)** 인챈트를 곡괭이에 부여하면 위 광석들을 **한 번에 대량 수확** — 채굴 효율의 핵심.

---

## 2. 광물 종류 (모드별)

### GeOre (지오어) — 정동석형 보석/금속 ⭐
바닐라 자수정 정동석과 같은 방식: **"움트는(budding) X 지오어"** 블록에서 **"X 지오어 무더기"** 가 자라며, 이를 캐서 보석/원료를 얻습니다. 거점에 두면 **재생 자원**이 됩니다.
- 일반: 석탄·구리·다이아몬드·에메랄드·금·철·청금석·레드스톤·석영·흑석영
- 금속: 알루미늄·납·니켈·오스뮴·백금·은·주석·텅스텐·아연·우라늄·모나자이트
- 보석: 루비·사파이어·토파즈
- 희귀: 고대 잔해·올더모디움·바이브라늄·언옵테이늄
- **georenouveau 탐지봉(Dowsing Rod)**: 사용 시 해당 지오어가 **벽 너머로 빛나 보임**(매직 파인드+점술). 지오어 골렘 부적도 추가.

### Forbidden Arcanus
- **비전 수정 광석** → 비전 수정(마법 재료), **경험석화 광석** → 경험치 관련.
- **데오룸**(고급 금속, **강화된 데오룸 곡괭이** 제작), **스텔라라이트 조각**(스텔라 아르카눔 블록에서).

### Ice and Fire *(한글 미적용 — 영문 표기)*
- **은 광석 / 심층암 은 광석** → 은 장비(드래곤에 추가 피해), **사파이어 광석** → 드래곤 제작.

### Deeper and Darker
- 깊은 어둠 차원의 **음울암(Gloomslate)·스컬크 돌** 변종 광석(석탄~다이아 8종) + 금속 **레조나리움**(도구/방어구).

### Tetra 재료 *(본체 한글 미적용)*
- Blue Skies 차원(에버브라이트/에버던)의 광물: 투휘석·차로아이트·월장석·파이로프·아쿠아이트·터키석·팔사이트·벤튬·호라이즈나이트 → 모듈형 도구 부품.

## 3. 채굴 도구

### Tetra 모듈형 곡괭이/망치 *(한글 미적용)*
- **곡괭이**와 **망치**가 핵심. 각각 **티어(tier)** 가 있어 "티어 N 곡괭이/망치 필요" 식으로 작동.
- **단조 망치(Forge Hammer)**: 모듈 제작·해체. 부착물로 효율↑·망치 티어↑.
- **기반암 추출기(Chthonic Extractor)**: 기반암을 망치질해 지하 광물 추출.
- **정동석(Geode)**: 워크벤치에서 깨면 광물, **행운(Fortune)** 영향받음.
- 특수 채굴 부품: **Stonecutter / Earthpiercer**("강력한 채굴 도구").

### Apotheosis 곡괭이 강화 *(본체 한글 미적용)*
- **방사형 채굴(Radial Mining)**: 키로 모드 전환 → 한 번에 여러 블록 채굴.
- **젬 소켓팅**: **대장장이 작업대**에서 곡괭이에 젬 장착(곡괭이/채굴 젬 클래스). 광석 채굴 시 추가 드롭(가공 금·암브로시움 파편·벤티움 등).

---

## 4. 광맥 채굴(Vein Mining) vs 나무 찍어내기(TreeChop)

### ⛏️ 광맥 채굴 (Vein Mining) — 광석용
- **도구에 "광맥 채굴" 마법부여가 있어야 작동**합니다. (없으면 "광맥 채굴을 하려면 도구에 §3광맥 채굴 마법부여가 필요합니다" 안내)
- **발동 방식** (설정에 따라):
  - **키 홀드**: 지정한 "광맥 채굴 활성화" 키를 누른 채 채굴 → 연결된 동일 블록을 함께 부숨. (옵션 > 조작 > 키 설정에서 키 지정)
  - **웅크리기**: 웅크리면 광맥 채굴(또는 웅크리면 정지 — 설정에 따라).
- 최대 블록 수는 기본값 + 마법부여 레벨당 증가. 내구도 소진 시 중단.

### 🪓 나무 찍어내기 (TreeChop) — 나무 전용
- **마법부여 불필요**(기본 제공). 나무를 **찍어내기(Chopping)** → 충분히 찍으면 **쓰러뜨리기(Felling)** 로 나무 전체가 넘어감.
- 단축키/오버레이로 모드 전환(찍어내기 전환, 웅크리기 동작 전환).

> 요약: **광석·광맥 = Vein Mining(인챈트 필요)**, **벌목 = TreeChop(기본 제공)** 으로 역할이 분리됩니다.

## 5. 광부(Miner) 직업 — Skill Tree

skilltree의 **광부 클래스**. **채굴 속도 + Apotheosis 젬 활용 + 동굴 생존력** 특화. (정확한 수치는 게임 내 노드 툴팁 확인)

### 주요 노드
| 계열 | 노드 |
|---|---|
| 기초 | **채굴 속도**(6단계), 마스터리 **탐욕(Greed)**, 빛의 수정 |
| 젬·제작 | **젬 증가**, **발굴(Excavation)**, 무기 내 젬 피해, 방어구 내 젬 위력, **연마/컬리넌/귀중한 무기** |
| 생존 | 방어력, **안전모**, **돌의 심장**, 최대 체력/**생명의 수정**, 생명 재생/**치유의 수정** |

### 하위 직업
- **탐험가(Explorer)** → 이동속도·신속·칠리화(Seven-League Boots). 동굴/탐험 특화.
- **보석세공사(Jeweler)** → 장신구 내 젬 위력·별 파편·부적. 젬/장신구 특화.

> 광부는 **젬을 무기·방어구·장신구에 박아 전투력으로 환산**하는 빌드입니다. 채굴이 곧 강화로 이어집니다.

## 6. MineColonies 광부 일꾼 *(한글 미적용 — 영문)*

직접 캐기 싫다면 **콜로니 광부**에게 맡깁니다.
- **광산(Mine) 건물**에 광부 배치 → 자동으로 갱도를 파며 광석·자갈 채집.
- **건물 레벨별 도구 티어 제한**: lv0 나무/금 → lv1 돌 → lv2 철 → lv3 다이아 → lv4~5 마법부여. 더 깊이 파려면 **오두막 업그레이드** 필요("Please upgrade my hut to dig deeper!").
- 호위병을 광산에 배치해 몹으로부터 보호 가능.

## 7. 추천 진행

1. **기초 채굴**: 곡괭이로 철·구리·다이아 확보. 곡괭이는 Tetra 모듈형 또는 강화 곡괭이로.
2. **광맥 채굴 세팅**: 곡괭이에 **광맥 채굴 마법부여** + **행운** → 광석 대량 수확. 발동 키 지정.
3. **젬 파밍**: 던전/광석에서 젬 수집 → **대장장이 작업대**로 곡괭이·장비에 소켓팅. **광부 직업**으로 젬 위력 강화.
4. **정동석 농장**: **움트는 지오어**를 거점에 모아 보석·금속 재생. **탐지봉**으로 위치 탐색.
5. **후반 자원**: Forbidden Arcanus(비전 수정·데오룸), Deeper Darker(레조나리움), Blue Skies(Tetra 재료)까지 확장.

## 8. 자주 막히는 부분

- **Q. 광맥 채굴이 안 돼요** → 도구에 **광맥 채굴 마법부여**가 있어야 하고, **활성화 키를 지정**해 눌러야 합니다(또는 웅크리기 설정).
- **Q. 광석/도구 이름이 영어예요** → Ice and Fire·Tetra·Apotheosis 본체·MineColonies는 한글 미적용 부분이 있습니다. 영문명을 참고하세요.
- **Q. 콜로니 광부가 더 깊이 안 파요** → 광산 건물을 **업그레이드**하세요(레벨별 채굴 깊이·도구 티어 제한).
- **Q. 젬을 어디에 박나요** → **대장장이 작업대**에서 소켓에 장착. 소켓이 없으면 **소켓의 인장**으로 추가(Apotheosis 강화는 `모드팩_플레이_가이드.md` 1장 참고).
