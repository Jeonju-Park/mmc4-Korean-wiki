# 🧰 생산성 툴 가이드 (제작법 포함)

> Medieval MC MMC4 — 보관·이동·수집·가공·거래를 편하게 해주는 도구 총정리
> 제작법은 모드 jar의 실제 레시피 데이터에서 추출했습니다. (3×3 = 제작대 배치)
> ⚠️ 일부 Quark/Supplementaries 기능은 모드팩 설정으로 꺼져 있을 수 있어 제작이 안 될 수 있습니다.

---

## 0. 무엇부터 만들까 (우선순위)

1. **여행자의 배낭** — 휴대 보관 + 제작/화로/자동수집 업그레이드. 탐험·채굴 지속력의 핵심.
2. **전송석(Waystones)** — 거점 간 순간이동. 왕복 시간 제거.
3. **톰의 보관 네트워크** — 상자 통합 검색·제작 터미널.
4. **제재소 / 광맥 채굴·나무 베기** — 자원 가공·수집 효율.
5. **거래소** — 주민 거래 통합.

> 표기: `판자`=아무 나무 판자, `나무상자`=상자, `철괴`=철 주괴, `(빈)`=빈 칸.

## 1. 🎒 여행자의 배낭 (Traveler's Backpack)

**이득**: 대용량 보관 + 유체 탱크 2개 + 침낭(즉석 리스폰) + **기능 업그레이드**(제작대·화로·자동수집·자석·자동급식)를 등에 지고 다님.

### 1-1. 배낭 탱크 (본체 재료)
```
유리  철괴  유리
유리  (빈)  유리
유리  철괴  유리
```

### 1-2. 기본 배낭
```
가죽   끈     가죽
탱크   나무상자  탱크
가죽   침낭    가죽
```
- 탱크 = 위에서 만든 배낭 탱크, 침낭 = 아무 색 침낭

### 1-3. 등급 배낭 (철/금/다이아) — 표준 배낭을 둘러서
```
(빈)  재료  (빈)
재료  표준배낭 재료
(빈)  재료  (빈)
```
- 철 = 철괴 / 금 = 금괴 / 다이아 = 다이아몬드 (각각 **표준 배낭**을 베이스로)
- **네더라이트 배낭**: 대장장이 작업대 → 네더라이트 주형 + **다이아 배낭** + 네더라이트 주괴

### 1-4. 빈 업그레이드 (모든 기능 업그레이드의 기반, 4개 산출)
```
(빈)  끈   (빈)
끈    가죽  끈
(빈)  끈   (빈)
```

### 1-5. 기능 업그레이드 (일반 제작대) — 핵심
| 업그레이드 | 배치 |
|---|---|
| **제작** | 세로: 제작대 / 빈업그레이드 / 나무상자 |
| **화로** | 세로: 화로 / 빈업그레이드 / 나무상자 |
| **고속 화로** | 세로: 고속화로 / 빈업그레이드 / 나무상자 |
| **훈연기** | 세로: 훈연기 / 빈업그레이드 / 나무상자 |
| **픽업(자동수집)** | 윗줄 `호퍼 빈업그레이드 호퍼` / 아랫줄 `레드스톤×3` |
| **자석** | `엔더진주 (빈) 엔더진주` / `레드스톤 빈업그레이드 청금석` / `레드스톤 철괴 청금석` |
| **자동 급식** | `황금당근 빈업그레이드 황금사과` / `레드스톤×3` |
| **소각(void)** | `용암양동이 빈업그레이드 용암양동이` / `레드스톤×3` |
| **탱크 추가** | 가로: `배낭탱크 빈업그레이드 배낭탱크` |

> 부착: 업그레이드를 **가죽을 틀로** 배낭과 함께 대장장이 작업대에 넣거나, 배낭 GUI 슬롯에 직접 장착.

## 2. 📦 톰의 간단한 보관 (Tom's Simple Storage)

**이득**: 여러 상자를 연결해 **하나의 터미널**에서 검색·정렬·꺼내기, 제작 터미널은 네트워크 재고로 바로 제작. AE2/RS보다 가볍고 초반 접근 쉬움.

### 2-1. 저장 터미널
```
판자   비교기  판자
나무상자 발광석  유리
판자   비교기  판자
```

### 2-2. 인벤토리 커넥터 (네트워크의 두뇌 — 상자에 부착)
```
판자   비교기  판자
나무상자 다이아  나무상자
판자   엔더진주 판자
```

### 2-3. 제작 터미널
```
제작대  다이아   제작대
다이아  저장터미널 다이아
제작대  다이아   제작대
```

### 2-4. 인벤토리 케이블 (6개 산출)
```
나무막대 판자   나무막대
나무뚜껑문 나무상자 나무뚜껑문
나무막대 판자   나무막대
```

### 2-5. 무선 터미널
```
판자   비교기  판자
망원경  발광석  유리
판자   엔더진주 판자
```
- **고급 무선 터미널**(차원 넘어 접근): 대장장이 작업대 → 네더라이트 주형 + 무선 터미널 + 네더라이트 주괴

**사용법**: ① 상자들을 붙이거나 케이블로 연결 → ② 한 상자에 **인벤토리 커넥터** 부착 → ③ 커넥터/케이블 끝에 **저장(제작) 터미널** 부착 → ④ 터미널 열어 검색·제작. 멀리 떨어진 상자군은 **케이블 커넥터**로 연결.

## 3. 🌀 전송석 (Waystones)

**이득**: 등록한 전송석 간 순간이동으로 왕복 시간 제거. 휴대형/주문서로 야외 귀환.

### 3-1. 전송석 (거점 텔레포트 지점)
```
(빈)   돌벽돌  (빈)
돌벽돌  휴대용전송석 돌벽돌
흑요석  흑요석  흑요석
```

### 3-2. 휴대용 전송석 (손에 들고 재사용, 쿨다운)
```
자수정  엔더진주 자수정
엔더진주 에메랄드 엔더진주
자수정  엔더진주 자수정
```

### 3-3. 공명석 (같은 색끼리 묶이는 전송 그룹)
```
돌벽돌  돌벽돌  돌벽돌
(빈)   휴대용전송석 (빈)
흑요석  흑요석  흑요석
```

### 3-4. 전송 가루 (무정형, 4개 산출)
- **엔더진주 1 + 자수정 1**

### 3-5. 전송판 (밟으면 연결 지점으로 자동 전송)
```
돌벽돌  전송가루 돌벽돌
전송가루 부싯돌  전송가루
돌벽돌  전송가루 돌벽돌
```

### 3-6. 주문서 (각 3개 산출)
- **귀환 주문서**(1회용, 지정 귀환 전송석으로): `금조각 보라색염료 금조각` / `종이 종이 종이`
- **전송 주문서**(아무 전송석으로 1회): `금조각 보라색염료 금조각` / `금조각 엔더진주 금조각` / `종이 종이 종이`
- **연결 주문서**(특정 전송석 연결): `보라색염료×3` / `금조각 엔더진주 금조각` / `종이×3`

**사용법**: 전송석 설치 시 자동 등록(우클릭 이름 지정). 다른 전송석/공명석 우클릭 → 목적지 선택 이동.

## 4. 🪚 제재소 (Sawmill)

**이득**: 원목→판자 가공 효율 증대(통나무 1개에서 더 많은 판자, 막대·울타리 가공). *(한글 미적용 — 게임 내 "Saw")*

### 제재소 블록
```
막대    막대    막대
나무울타리 철괴   나무울타리
나무반블록 나무반블록 나무반블록
```
**사용법**: 블록 우클릭 → GUI에 원목/목재 넣고 가공물 선택.

## 5. 🏪 거래소 (Trading Post)

**이득**: 주변 주민들의 거래를 **한 GUI에서 통합** — 여러 주민을 돌 필요 없이 원격 일괄 거래.

### 거래소 블록
```
(빈)  에메랄드 (빈)
판자   판자   판자
막대   (빈)   막대
```
**사용법**: 설치 후 우클릭 → 범위 내 주민 거래 항목이 한 화면에 모임, 인벤토리 재고로 바로 거래.

## 6. ⛏️🪓 채굴·벌목 효율 (Vein Mining / TreeChop)

- **광맥 채굴(Vein Mining)**: 제작 없음. 도구에 **"광맥 채굴" 인챈트** 부여 후 키/웅크리기로 발동 → 연결된 동일 광석 일괄 채굴.
- **나무 베기(TreeChop)**: 제작 없음. 나무 밑동을 베면 점진적으로 깎여(Chopping) **나무 전체가 쓰러짐(Felling)** → 공중 통나무 줍기 불필요. 설정 GUI에서 모드 토글.

> 자세한 내용은 `광질_가이드.md` 4장 참고.

## 7. 🔧 Supplementaries 편의 블록 (제작법)

> ⚠️ 모드팩 설정에 따라 일부 비활성일 수 있음.

- **도르래(Pulley)** — 밧줄/사슬 자동 권취(우물·엘리베이터식)
  ```
  판자   나무반블록 판자
  나무반블록 철괴   나무반블록
  판자   나무반블록 판자
  ```
- **크랭크(Crank)** — 손으로 돌려 레드스톤 신호: `(빈) 나무막대 (빈)` / `돌 돌 돌`
- **유리병(Jar)** — 아이템/액체 대량 보관·전시:
  ```
  유리  나무반블록 유리
  유리  (빈)   유리
  유리  유리   유리
  ```
- **아이템 선반(Item Shelf)** — 벽면 전시(2개): 가로 `나무반블록 나무반블록 나무반블록`
- **금고(Safe)** — 소유자 잠금 보관(철괴+나무상자 기반, 이름표로 잠금). *특수 레시피*
- **동물 우리(Cage)** — 소형 몹 포획·운반: `철괴×3` / `철창 (빈) 철창` / `나무반블록×3`

## 8. 💎 유물(Relics) — 편의 아이템 (제작 불가, 전리품)

> ⚠️ Relics 아이템은 **제작 레시피가 없습니다.** 던전·구조물 상자 등에서 **드랍**되며, 사용으로 경험치를 쌓아 레벨업·강화합니다.

생산성/편의 계열:
- **매직 미러** — 마지막 안전 지점/스폰으로 귀환(복귀 이동 절약)
- **엔더 핸드** — 엔더 진주 소모 없는 원거리 이동/투척
- **무한한 햄** — 재충전 식량(식량 관리 부담 제거)
- **화살통** — 화살 대량 보관·자동 공급
- **겉날개 부스터** — 겉날개 추진(폭죽 대체)
- **아쿠아/마그마 워커** — 물/용암 위 보행
- **해파리 목걸이** — 수중 호흡/이동, **말 피리·포자 주머니** 등

## 9. 📚 Quark 편의 (제작법) — *설정 활성 시*

> ⚠️ Quark 기능은 설정 플래그로 켜고 끕니다. 모드팩에서 꺼졌으면 제작 불가.

- **운송용 상자(Crate)** — 정렬 없이 빠르게 넣고 빼는 보관함:
  ```
  철괴  판자  철괴
  판자  나무상자 판자
  철괴  판자  철괴
  ```
- **씨앗 주머니(Seed Pouch)** — 씨앗 대량 보관 + 일괄 심기: `(빈) 끈 (빈)` / `가죽 씨앗 가죽` / `(빈) 가죽 (빈)`
- **구리 파이프(Pipe)** — 아이템 이송(6개): 세로 `구리괴 / 유리 / 구리괴`
- **배낭(Backpack)** — 추가 인벤토리(레비저 가죽 필요): `가죽 레비저가죽 가죽` / `가죽 나무상자 가죽` / `가죽 철괴 가죽`
- 그 외 **상자 자동 정렬, Shift 더블탭 즉시 보관** 등 레시피 없는 편의 기능(설정 활성 시 자동 동작).

## 10. 추천 셋업 순서

1. **초반**: 여행자의 배낭(+탱크) → 빈 업그레이드 → **제작/화로 업그레이드** 부착(이동 중 제작·제련).
2. **거점**: 톰의 보관 네트워크(인벤토리 커넥터 + 저장/제작 터미널)로 창고 통합.
3. **이동망**: 주요 거점마다 **전송석** + 휴대용 전송석/귀환 주문서.
4. **수집 효율**: 곡괭이에 **광맥 채굴** 인챈트, **나무 베기**로 벌목, **제재소**로 목재 가공.
5. **자동 수집**: 배낭 **픽업/자석 업그레이드**로 채굴 중 자동 회수.
6. **거래**: **거래소**로 주민 거래 통합, 유물(매직 미러 등)은 던전에서 수급.

## 11. 자주 막히는 부분

- **Q. Quark/Supplementaries 아이템이 제작 안 돼요** → 모드팩 설정 플래그로 꺼졌을 수 있습니다(정상). 다른 보관/이송 수단을 쓰세요.
- **Q. 유물(Relics)을 못 만들어요** → 제작 불가. **던전 상자**에서 얻습니다.
- **Q. 보관 터미널이 텅 비어요** → 상자에 **인벤토리 커넥터**가 부착됐는지, 터미널이 커넥터/케이블에 연결됐는지 확인.
- **Q. 배낭 업그레이드를 어떻게 끼우나요** → 가죽을 틀로 대장장이 작업대에 배낭과 함께, 또는 배낭 GUI 슬롯에 장착.
