# 🛡️ 근딜 탱커 종결 빌드 가이드 (제작법·재료·작업대 총정리)

> "방패 들고 때리는" 근접 탱커(브루저)를 종결까지 키우는 단계별 장비 가이드.
> 방어구·무기·방패·장신구·젬·인챈트로 나눠 **제작법 / 효과 / 필요 작업대·도구 / 재료 / 작업법**을 정리했습니다.
> 모드 jar의 실제 레시피 데이터 기반. (수치·한글 미적용 항목은 JEI로 확인 권장)

## 빌드 개요
- **직업**: 대장장이(Blacksmith) → 병사(Soldier) — "방패 장착 시 피해" 노드로 방패 들고도 딜. (방어=Blocking은 방패 필수)
- **핵심 키워드**: **체력 + 방어 + 흡혈 + 막기**. "안 죽고, 때리며 자가회복"이 탱커의 본질.
- **진행 축**: 던전 접사템 → Apotheosis 강화(젬·소켓·인챈트) → Cataclysm 보스 세트 → Ice&Fire 드래곤스틸.

## 1. 🛡️ 방어구 (Armor)

### 단계 요약
| 단계 | 추천 | 비고 |
|---|---|---|
| 초반 | 철 → 다이아 | 바닐라 |
| 중반 | 네더라이트 / **드래곤비늘 갑옷** | 브레스 저항 |
| 후반 | **드래곤스틸 세트** | 용광로 필요 |
| 종결 | **커스뮴 세트(자동 부활)** / 이그니티움 | Cataclysm 보스 |

### ⭐ 종결 — Cataclysm 커스뮴 / 이그니티움 (스미싱)
둘 다 **네더라이트 방어구 + 전용 형판 + 보스 주괴**로 대장장이 작업대(스미싱)에서 강화.

**① 전용 형판 제작 (제작대 3×3)**
- **커스뮴 형판**:
  ```
  금주괴  금주괴  금주괴
  흑강주괴 네더라이트형판 흑강주괴
  금주괴  금주괴  금주괴
  ```
- **이그니티움 형판**:
  ```
  네더벽돌 블레이즈가루 네더벽돌
  블레이즈가루 네더라이트형판 블레이즈가루
  네더벽돌 블레이즈가루 네더벽돌
  ```

**② 스미싱(대장장이 작업대)**: `네더라이트 부위 + 형판 + 주괴` → 커스뮴/이그니티움 부위 (투구·흉갑·레깅스·부츠 각각)

**③ 주괴 입수 (보스 드롭 = 종결 콘텐츠)**
- **커스뮴 주괴** ← **말레딕투스(Maledictus)** 처치
- **이그니티움 주괴** ← **이그니스(Ignis)** 처치
- 풀세트 ≈ 주괴 24개 → 보스 반복 처치 + 네더라이트 풀셋

**세트 효과 (탱커 관점)**
| 부위 | 커스뮴(생존형 ⭐) | 이그니티움(공격·지형형) |
|---|---|---|
| 흉갑 | **사망 시 체력5로 부활 + 5초 무적** (이후 6분 디버프) | 겉날개 결합 가능 |
| 투구 | 벽 뒤 적 발광 | 근처 적 화염 낙인 |
| 레깅스 | 낮은 확률 공격 회피 | 화염 반사 |
| 부츠 | 낙하피해↓ + 백스텝 | 용암 위 걷기 |

> **탱커 1순위 = 커스뮴 흉갑(자동 부활)**. 네더/용암 지형 위주면 이그니티움.

### 후반 — Ice & Fire 드래곤스틸 / 드래곤비늘
- **드래곤스틸 주괴**: **드래곤 용광로(Dragonforge)** 에서 `철 주괴 1 + 속성 드래곤의 피 1` → 1개 (작업대 아님, 멀티블록 건설 필요)
  - 드래곤의 피 = **빈 병 들고 죽은 드래곤 사체 우클릭**
- **드래곤스틸 방어구**: 바닐라 배치 + 드래곤스틸 주괴 → 게임 내 최상위(브레스 저항)
- **드래곤비늘 갑옷(중간 단계)**: 바닐라 배치 + **드래곤 비늘 8개/부위** → "드래곤 브레스 방어력↑". 용광로 불필요라 진입 쉬움.

## 4. 💎 장신구 (Curios / Relics)

### 불사의 부적 = 토템을 Curios에 ⭐
- **별도 제작 없음.** 바닐라 **불사의 토템**을 **Curios의 "Charm" 슬롯**에 넣으면, **손에 안 들어도(방패·무기 유지) 사망 시 자동 부활**.
- 토템 입수: 약탈자 습격대장 처치.

### Relics — 방어/생존 유물 (전부 제작 불가, 던전 전리품)
| 유물(한/영) | 슬롯 | 효과 |
|---|---|---|
| **반사 목걸이 / Reflection Necklace** | 목걸이 | 받은 피해 축적 후 흑요석 폭발 + 기절 (리벤지 탱커 ⭐) |
| **성스러운 로켓 / Holy Locket** | 목걸이 | 충전 소모 시 **일시 무적** |
| **포자 주머니 / Spore Sack** | 등 | 피격 시 포자 → 적 접근 폭발(중독·둔화·치유차단) |

## 5. 💠 젬 (Apotheosis) — 탱커의 핵심

### 시스템 흐름
1. **소켓 뚫기**: **소켓의 인장(Sigil of Socketing)** 을 제작대에서 장비와 함께 → 소켓 추가
2. **젬 등급업**: **보석 절단 작업대(Gem Cutting Table)** 에서 `보석 가루 + 해당 등급 희귀 재료`
3. **젬 장착**: **바닐라 대장장이 작업대(Smithing Table)** 에서 장비 + 젬 → 소켓에 삽입
4. **재료 수급**: **분해 작업대(Salvaging Table)** 에서 접사템 분해 → 희귀 재료

### 필요 작업대 제작
- **보석 절단 작업대**: `매끄러운돌 가위 매끄러운돌 / 판자 보석가루 판자 / 판자 (빈) 판자`
- **분해 작업대**: 구리3 + 철곡괭이/도끼 + 용암양동이 + 대장장이 작업대 + 보석가루2
- **소켓의 인장(3개 산출)**: `보석가루 드래곤숨결 보석가루 / 점판암 점판암 점판암 / 보석가루 자수정 보석가루` (점판암=보석 융합 점판암)

### 탱커용 젬 (효과: 흔함→고대 순)
| 젬(한/영) | 부위 | 효과 |
|---|---|---|
| **수호자 / Guardian** | 흉갑·레깅스 | 방어력↑ / 방패 방어력 +5~45% |
| **싸움꾼 / Brawlers** | 흉갑·레깅스 | 최대 체력 +1~10 / 방패 체력↑ |
| **안정함 / Ballast** | 흉갑·레깅스 | **넉백 저항**(합 ≥1.0 = 면역) |
| **투사 / Combatant** | 흉갑·레깅스 | **물리 피해 감소 5~27.5%** |
| **대지 / Earth** | 흉갑·레깅스 | **보호 인챈트 +1~4** |
| **피의 군주 / Blood Lord** ⭐ | 검·흉갑 | **체력 흡수 +5~75%**, 막을 시 회복 (흡혈 탱커 핵심) |

### 희귀 재료 등급 (분해 작업대에서 해당 등급 접사템 분해)
신비한 고철(흔함) → 낡은 원단(드뭄) → 빛나는 수정 파편(희귀) → 아르카나 모래(영웅) → 신이 단조한 진주(신화) → 무한의 발현(고대) + 공통 **보석 가루**(보석에 모루 떨궈 획득)

> 탱커 세팅: **방어구 = 수호자+싸움꾼+안정함+투사 / 검 = 피의 군주(흡혈)**. 완벽한(고급) 등급까지 보석 절단 작업대로 올리세요.

## 6. ✨ 인챈트 (Apotheosis / Apotheotic Additions)

| 인챈트(한/영) | 효과 | 탱커 가치 |
|---|---|---|
| **반사적인 방어 / Reflective** | 막으면 공격자에게 피해 | 방패 핵심 ⭐ |
| **방패 치기 / Shield Bash** | 방패로 피해 가능 | 방패 |
| **리바운딩 / Rebounding** | 근접 피격 시 공격자 강하게 밀침 | 방어구 |
| **광전사의 격분 / Berserker's Fury** | 피격 시 격분 버프 | 피격 발동 |
| **생명 수선 / Life Mending** | 받은 회복을 흡수해 내구도 수리 | 흡혈 빌드 ⭐ |
| **얼어붙은 가시 / Icy Thorns** | 공격자 둔화 | 가시형 |
| 보호 / 마법 보호 | 피해 감소 | 기본 방어 |

### ⚔️ 근접 무기 공격 인챈트 (딜·흡혈·격분)
> 이 모드팩은 **언데드(좀비·스켈레톤·소환수)가 압도적**이라, 실전 데미지 1순위는 **강타(Smite)** 입니다.

**① 데미지 코어 — 무기당 택1 (서로 충돌)**
- **강타(Smite)** ⭐ — 언데드 다수라 실전 최강
- **날카로움(Sharpness)** — 범용 (언데드 아닌 보스 상대)
- 상황용: 우민의 파멸(습격)·스컬크 강타(딥다크/워든)·진압(콜로니 습격자)

**② 자가회복 (탱커 핵심)**
- **생명 수선(Life-Mending)** — 받은 회복을 흡수해 무기 내구도 수리(흡혈·재생 시너지)
- **생명 움켜쥐기(Life Grasp, Dungeons Delight)** — 타격 시 회복(라이프스틸)

**③ 탱커 시너지**
- **광전사의 격분(Berserker's Fury)** — **피격 시 공격력↑ 버프** (맞을수록 강해짐)
- **분할(Splitting)** — 타격 시 **주변 적에게도 피해(광역 클리브)**

**④ 부가**: 약탈(Looting) · 발화(Fire Aspect) · 포획(Capturing)/스캐빈저(Scavenger)

**🏆 추천 검 세팅**
```
강타 + 생명 수선 + 생명 움켜쥐기 + 광전사의 격분 + 분할 + 약탈 + 발화
```
> ⚠️ **강타/날카로움/우민의 파멸/스컬크 강타는 택1**(충돌). 나머지는 함께 가능. 무기에 **피의 군주 젬(흡혈)** 까지 더하면 자힐이 폭발적.

### 마법 부여대 강화
- **에테르나(최대 레벨)·퀀타(부정효과↓)·아르카나(희귀 마법·부여 수↑)** 3능력치를 **책장**으로 향상.
- 책장: 지옥/바다/심층/엔드 → **주입(infused)** 단계로 강화.
- **마법 부여 문고(Enchantment Library)**: `엔더상자 주입책장 엔더상자 / 주입책장 마법부여대 주입책장 / 엔더상자 주입책장 엔더상자` — 마법책 저장/추출.

> 탱커 인챈트 우선순위: **방패 = 반사적인 방어 + 방패 치기**, **방어구 = 보호 + 리바운딩**, **검 = 생명 수선(흡혈 연계)**.

## 🎯 종결 탱커 세팅 (한눈에)
1. **방어구**: 커스뮴 풀세트(자동 부활) → 각 부위 **수호자+싸움꾼+안정함+투사** 젬 + 보호·리바운딩 인챈트
2. **무기**: 한손 검(드래곤스틸/카타나/흑강) + **피의 군주 젬(흡혈)** + 생명 수선 인챈트 (또는 수호의 건틀릿으로 어그로)
3. **방패**: 불꽃의 보루 + **반사적인 방어 + 방패 치기**
4. **장신구**: Charm 슬롯에 불사의 토템 + 반사 목걸이·성스러운 로켓·마그마 워커
5. **직업**: 대장장이 병사(방패 장착 시 피해 + 방어·체력)

## 📦 종결 재료 백과 (위치·제작법 상세)

> "흑강·드래곤스틸·이그니티움·정령 재료·위더본·젬을 대체 어디서 어떻게?" 를 파일 근거로 정리했습니다.

### ⚫ 흑강 (Black Steel) — Cataclysm
**제작 불가, 드롭/상자 전용.**
- **서리 감옥(Frosted Prison) 보물상자** — 1상자당 **5~9개**(최고 공급원)
- **드라우그르 / 정예 드라우그르 / 왕실 드라우그르** 처치 — 2.5% 확률
- **아프트강그(Aptrgangr)** 처치 — 1~2개 확정
- 압축: 주괴 9 ↔ 블록 1, 주괴 1 ↔ 조각(너겟) 9, 조각 9 → 주괴 1 (조합대)
> 루트: **서리 감옥 구조물 탐험 + 드라우그르 사냥**. 커스뮴 형판에 **흑강 주괴 2개**가 필요해 선행 재료.

### 🔥 이그니티움 주괴 (Ignitium) — Cataclysm
- **이그니스(Ignis) 처치** → **주괴 3개 확정**(유일 입수처)
- 이그니스 소환: **타오르는 잿더미를 제단에 바쳐** 소환(네더 종결 보스)
- 압축: 주괴 9 ↔ 블록 1

### 💀 커스뮴 주괴 (Cursium) — Cataclysm
- **말레딕투스(Maledictus) 처치** → **주괴 3~4개**(유일 입수처)
- 소환: **묘비를 눌러** 소환
- 압축: 주괴 9 ↔ 블록 1

### 🌊 라크리마 (Lacrima) — Cataclysm (해양 재료)
- **스킬라(Scylla) 처치** 8~16개(최대) / 클라우디안 5~8 / 신다리아·히포캄투스 1~4
- **아크로폴리스(Acropolis) 보물상자** 3~5개

### 🔨 스미싱(Smithing) 작동 방식 — 대장장이 작업대
**형판 + 베이스 + 추가재료** 조합:
- 이그니티움/커스뮴 방어구 = **형판 + 네더라이트 부위 + 해당 주괴**
- **형판 제작(조합대, 복제 아님)**:
  - 이그니티움 형판: `네더벽돌·블레이즈가루` 테두리 + 가운데 **네더라이트 강화 형판**
  - 커스뮴 형판: `금괴` 위/아래 + `흑강주괴` 양옆 + 가운데 **네더라이트 강화 형판**
  > Cataclysm 형판은 바닐라식 다이아 복제가 **아니라** 네더라이트 형판을 소모해 1개씩 제작.

### 🐉 드래곤스틸 (불/얼음/번개) — Ice and Fire
**전용 멀티블록 "드래곤 용광로(Dragonforge)" 필요** (조합대 아님).
- **주괴 제련**: 용광로에 **철 주괴 + 속성 드래곤의 피** → 드래곤스틸 주괴 1 (1000틱)
- **드래곤의 피**: **빈 유리병 들고 죽은 드래곤 시체 우클릭**
- **용광로 건설**:
  1. **드래곤 비늘 블록** (드래곤 비늘 9 → 블록 1, 비늘은 살아있는 드래곤 처치 드롭)
  2. **용광로 벽돌**: `돌벽돌 + 드래곤 비늘 블록` → 4개
  3. **용광로 코어**: 벽돌 8 + 가운데 **드래곤 심장**(시체에서)
  4. 입력구·3층 구조로 조립
  5. **가동**: 입구 앞에 **사슬로 묶거나 길들인 드래곤**을 두면 숨결로 동력 공급
- 압축: 주괴 9 ↔ 블록 1 (3속성 동일)
> ⚠️ 무기에 **위더본 손잡이** 추가 필요(아래).

### 🦴 위더본 / 시들어버린 뼈 (Witherbone) — Ice and Fire
**제작 불가, 드롭/상자.** (일반 위더 스켈레톤 아님!)
- **묘소(Mausoleum) 구조물 상자** — 1~8개(최고 공급원)
- **코카트리스(Cockatrice)** 처치 — 0~2개
- 가공: 위더본 1 → 시든 뼈 조각 3 (조합대)

### 👹 정령 카타나 재료 — Samurai Dynasty
**한손 카타나(방패 병행)**. 재료는 정령 몹 드롭:
| 재료 | 드롭 몹 |
|---|---|
| 오니 뿔 | **오니(Oni)** (+루비) |
| 키츠네 꼬리 | **키츠네/두 꼬리 여우** |
| 오니비 불꽃 | **오니비(Onibi)** |
| 조로구모 눈 | **조로구모** |
| 아카나메 혀 | **아카나메** |
- **베이스 체인**: 카타나 손잡이(막대+금너겟+가죽) → 카타나(강철2+금2+손잡이) → 네더라이트 카타나(스미싱)
- **정령 형판**: `정령석 벽돌` 테두리 + 다이아 + 기존 형판 → **2개 복제**. (정령석 벽돌 = 돌 8 + **오니비 불꽃** → 8개)
- **정령 카타나**: 네더라이트 카타나 + 정령 형판 + 해당 정령 재료 스미싱
> ⚠️ 정령 형판 **최초 1개**의 입수처는 파일 미확인(구조물 드롭 추정). 1개만 구하면 무한 복제.

### 💠 젬 & 보석 가루 (Apotheosis)
- **젬 드롭**: **모든 몹 처치/던전 상자**에 전역 적용되는 **어포틱 보상(affix_loot)** 으로 확률 드롭. (차원별로 나오는 젬이 다름)
- **보석 가루(Gem Dust)**: **분해 작업대(Salvaging Table)** 에서 **젬·접사템을 분해** → 가루 획득(희귀 등급일수록 더 많이). 또는 **보석에 모루를 떨궈** 획득.
- **소켓 흐름**:
  1. 보석 가루 확보 → **융합 점판암**(심층암8+가루1) → **소켓의 인장**(가루4+융합점판암3+드래곤숨결1+자수정1)
  2. **소켓의 인장**으로 장비에 소켓 추가(**최대 3개**)
  3. **대장장이 작업대**에서 장비+젬 → 소켓 삽입
  4. **보석 절단 작업대**에서 `보석 가루 + 등급별 희귀 재료`로 젬 **등급업**(흔함→고대)
