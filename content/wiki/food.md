## 1. 개요 — 요리 생태계

이 모드팩의 요리는 **Farmer's Delight(농부의 기쁨)** 를 기반으로 합니다. 농부의 기쁨이 **조리기구·작물·효과(편안함/배부름)** 라는 뼈대를 제공하고, 그 위에 **차원·테마별 확장 모드**들이 각자의 별미를 얹는 구조입니다.

| 모드 | 영역 | 특징 |
|---|---|---|
| **Farmer's Delight** (농부의 기쁨) | 공통 기반 | 조리용 냄비·도마·프라이팬·새 작물·효율 효과 |
| **Delightful** (딜라이트풀) | 확장 식재료 | 베리류·멜론·차/라떼·아이스크림 다수 추가 |
| **Dungeons Delight** (던전스 딜라이트) | 던전/몬스터 | 몬스터·구조물 재료 요리, 식칼형 무기(클리버) |
| **Twilight Delight** (트와일라잇 딜라이트) | 황혼의 숲 | 미노타우르스·사슴·히드라·가스트 요리 (한글 O) |
| **My Nether's Delight** (네더스 딜라이트) | 네더 | 호글린·가스트·네더 작물 요리 |
| **Ender's Delight** (엔더스 딜라이트) | 엔드 | 코러스·엔더맨·셜커 요리 |
| **Ocean's Delight** (오션스 딜라이트) | 바다 | 가디언·복어·오징어 해산물 요리 |
| **Aether Delight** (에테르 딜라이트) | 에테르 | 암브로시아·블루베리·메추라기 요리 |
| **Quark Delight** (쿼크 딜라이트) | 동굴/쿼크 | 게·이끼·이국 과일 요리 |
| **Brewin' and Chewin'** (브루인 앤 츄인) | 발효·음료 | 술통(Keg) 발효 — 술·치즈·김치·육포 |

> 💡 요리사 직업과 직결됩니다. **직접 만든 음식**이 직업 보너스를 발동시킵니다 → `2_직업_가이드.md` 참고.

## 2. 🍳 조리기구

농부의 기쁨은 4가지 조리 방식 + 발효(술통)를 제공합니다. **달구려면 아래에 열원(불)이 필요**한 기구가 많습니다.

| 조리기구 | 한글명 | 용도 | 사용법 |
|---|---|---|---|
| **Cooking Pot** | 조리용 냄비 | **스튜·수프·죽 등 그릇 음식** | 아래에 난로/모닥불 등 열원을 두고 재료 투입 → 끓이면 1인분씩 완성. ⭐효율 효과(편안함/배부름) 음식이 거의 전부 여기서 나옴 |
| **Stove** | 난로 | **열원 + 직화 구이대** | 위에 올린 음식(슬라이스 고기 등)을 자동으로 굽고, 냄비·프라이팬의 열원 역할도 함 |
| **Skillet** | 프라이팬 | **휴대용 가열** | 들고 다니며 열원 근처에서 다른 손에 재료를 들고 사용. 웅크려 블록으로 설치 가능. 침수/수중에서는 사용 불가 |
| **Cutting Board** | 도마 | **재료 손질(전처리)** | 도구(식칼 등)를 손에 든 채 우클릭 → 다진 고기·슬라이스·반죽 생성. 칼이 아니면 "다른 도구가 필요" 메시지 |
| **Campfire / Soul Campfire** | 모닥불 | **단순 가열** | 슬라이스 고기·생선·계란을 굽는 기본 화력. 버프 없음 |
| **Keg** | 술통 (의역) | **발효** (Brewin' and Chewin') | 재료를 넣고 발효. **온도 조건** 필요 — 냉각/가열 블록을 옆에 두어 조절 |

### 냄비 사용 팁
- 냄비 툴팁에 **"아래에 열원이 필요합니다"** 가 뜨면 불을 켜야 합니다. 달궈지면 **"달궈짐"** 표시.
- 완성된 그릇 음식은 보통 **그릇(Bowl)이 있어야 꺼낼 수 있습니다.** (잔치 음식은 별도 용기 필요)
- 식칼은 도마용 도구이자 가벼운 무기 — **잔디/밀/쌀을 베면 지푸라기**, 동물에게서 **추가 전리품·햄**을 얻습니다.

> 도마 심화 사용법(슬라이스/반죽 종류)·요리사 직업 시너지는 `website/content/detail/cook.md`(요리사 심화) 참고 — 중복 설명은 그쪽에 정리되어 있습니다.

## 3. 🌱 작물·식재료 (신규 작물)

농부의 기쁨은 **4종 신규 작물**(토마토·양배추·양파·쌀)을 추가하고, 기존 작물의 **야생 변종**도 자연 생성합니다.

### 신규 작물 재배법

| 작물 | 씨앗/모종 | 재배 방법 |
|---|---|---|
| 🍅 **토마토 (Tomato)** | 토마토 씨앗 | 일반 경작지. **밧줄(Rope)을 위에 걸면 덩굴이 더 높이 자라** 수확량↑ ("덩굴 토마토") |
| 🥬 **양배추 (Cabbage)** | 양배추 씨앗 | 일반 경작지에서 재배. 양배추 잎으로 추가 가공 |
| 🧅 **양파 (Onion)** | 양파 자체 (당근처럼) | 경작지에 양파를 직접 심음 |
| 🌾 **쌀 (Rice)** | 벼 / 쌀 | **얕은 물 웅덩이에 심어야 함** (벼는 물 밖에서 잘 못 자람). 자라면 윗부분이 벼(rice panicle) |

> 🍅 **썩은 토마토(Rotten Tomato)** 는 투척물 — 습격자(Raider)에게 던질 수 있습니다.

### 🌿 야생 작물 발견 위치 (생물군계별)

야생 작물은 자연에 자라며, 채집해 씨앗/식재료를 얻습니다.

| 야생 작물 | 발견 생물군계 |
|---|---|
| 야생 양배추 (Wild Cabbage) | **해변 해안가** |
| 야생 비트 (Sea Beet) | **해변 해안가** |
| 야생 당근 (Wild Carrot) | **평원·숲** 등 온대 지역 |
| 야생 양파 (Wild Onion) | **평원·숲** 등 온대 지역 |
| 야생 감자 (Wild Potato) | **산·타이가** 등 추운 지역 |
| 야생 토마토 (Tomato Shrub) | **사바나·사막** 등 건조 지역 |
| 야생 벼 (Wild Rice) | **늪지·정글**의 웅덩이 등 습한 지역 |

> 🍄 **버섯 군집(Mushroom Colony)**: **기름진 흙(Rich Soil)** 에서 완전히 자란 군집을 가위질하면 얻습니다. 기름진 흙은 **유기농 퇴비(Organic Compost)** 가 시간이 지나며 분해되어 생기며, 햇빛·물·인접 버섯이 분해를 촉진합니다.

### 차원별 확장 식재료 (예시)
- **Delightful**: 잎블록을 식칼로 쳐서 얻는 **녹차잎(Green Tea Leaf)**, 칸탈루프 멜론, 살몬베리(야생 덤불), 매치(말차) 가루
- **My Nether's Delight**: 진홍빛 숲의 **Powdery Canes(가루 줄기)** — 접촉 시 폭발하는 베리. 붉은 줄기를 잘라 채집
- **Aether Delight**: **Ambrosia(암브로시아)** 계열 신성한 식재료

## 4. ⭐ 효율 요리 표 (편안함 / 배부름)

농부의 기쁨에는 **그릇/접시 음식에만 붙는 두 강력 효과**가 있습니다. 둘 다 직접 만든 냄비 음식에서 나옵니다.

| 효과 | 한글 | 효과 내용 |
|---|---|---|
| **Comfort** | 편안함 | **배고픔과 무관하게 지속적으로 생명력 회복** (따뜻한 그릇 음식) |
| **Nourishment** | 배부름 | **(체력 회복 시 제외) 배고픔이 줄지 않음** = 오래 안 배고픔 (접시 음식) |

> 효과 정의(파일 원문): 배부름 = "생명력을 회복할 때를 제외하고 배고픔 수치가 줄지 않음", 편안함 = "배고픔 수치와 상관없이 생명력을 회복".

### 🥇 톱티어 (포만 14 / 효과 300초=5분)

| 효과 | 추천 음식 |
|---|---|
| **편안함 (Comfort) 5분** | 닭고기 수프 · 볶음밥 · 호박 수프 · 대구 스튜 · 우동(누들 수프) |
| **배부름 (Nourishment) 5분** | 구운 닭 · 꿀 바른 햄 · 셰퍼드 파이 · 속 채운 호박 · 양고기 구이 · 잡채 · 오징어 먹물 파스타 |

### 🍯 특수 효과 음료/음식
| 음식 | 효과 |
|---|---|
| 사과주 (Apple Cider) | 이동속도 증가 60초 |
| 발광 열매 커스터드 | 점프 강화 |
| 수박 주스 (Melon Juice) | 즉시 소량 생명력 회복 |
| 우유 병 (Milk Bottle) | 효과 1개 제거 |
| 핫 코코아 (Hot Cocoa) | **해로운 효과 1개 제거** |
| 믹스/과일 샐러드 | 재생 5초 |

> 효율 음식의 상세 레시피·요리사 빌드 활용은 `cook.md` 8~9장에 정리되어 있습니다(중복 방지 — 링크로 갈음).

## 6. 🍺 음료·발효주 (Brewin' and Chewin')

미번역 모드입니다. **술통(Keg)** 에 재료를 넣고 **온도 조건**을 맞춰 발효합니다.

### 발효 메커니즘 (술통)
- **온도 단계**: Cold(차가움) · Chilly(서늘함) · Normal(보통) · Warm(따뜻함) · Hot(뜨거움)
- 레시피마다 요구 온도가 다름 → **냉각 블록(얼음 상자 등)** 또는 **가열 블록(Heating Cask)** 을 술통 옆에 설치해 조절
- 잘못된 온도면 "Incorrect Temperature(온도 부적합)" 로 발효 불가

### 🍶 발효주 (음용 시 Tipsy 등 효과)
| 술 | 의역 |
|---|---|
| Beer / Mead / Vodka | 맥주 / 벌꿀술 / 보드카 |
| Rice Wine | 청주(쌀술) |
| Strongroot Ale | 뿌리 에일 |
| Steel-Toe Stout | 스틸토 스타우트(흑맥주) |
| Saccharine Rum / Red Rum | 단 럼 / 붉은 럼 |
| Egg Grog / Dread Nog | 에그 그로그 / 드레드 노그 |
| Kombucha | 콤부차 |
| Bloody Mary | 블러디 메리 |

### 🍻 술 관련 효과
| 효과 | 의역 | 내용 |
|---|---|---|
| **Tipsy** | 취기 | 화면이 흔들림. Lv3+ 마우스에 끌림(드래그), Lv4+ 채팅/표지판 글자가 뒤섞임. **피해가 지연**될 수 있음(주황 하트로 표시) |
| **Intoxication** | 만취 | 음식에서 **포만감(Saturation)을 얻지 못함** |
| **Raging** | 격노 | 근접 타격 시 **공격속도 +5% (최대 4중첩)** |
| **Sweet Heart** | 단맛의 마음 | 모든 허기 수치에서 포만감 기반 회복이 추가됨 |

> ⚠️ Dread Nog는 **불길한 징조(Bad Omen) +1** 부여 — 주의.

### 🧀 발효 음식 (술 외)
- **Kimchi** (김치), **Jerky** (육포), **Pickled Pickles** (피클), **Kippers** (훈제 청어)
- **Flaxen / Scarlet Cheese** (치즈) — 발효로 숙성, 피자·퐁뒤·키슈 등에 사용
- **Cocoa Fudge** (코코아 퍼지), **Creamy Onion Soup** (크림 양파 수프)

### ☕ 무알코올 음료 (Farmer's Delight + Delightful)
- **핫 코코아** (해로운 효과 제거), **사과주** (이속), **수박 주스**, **우유 병**
- Delightful 차/라떼: **Matcha Latte** (말차 라떼), **Azalea Tea** (진달래 차), **Lavender Tea** (라벤더 차), **Prickly Pear Juice** (선인장 주스), **Salmonberry/Source Berry Milkshake** (베리 밀크셰이크)

## 7. 🍽️ 초보 식단 추천

1. **시작**: 식칼(부싯돌 식칼)부터 제작 → 잔디를 베어 지푸라기 확보, 동물에서 추가 고기. 모닥불로 슬라이스 고기 구워 끼니 해결.
2. **냄비 도입**: 조리용 냄비 + 난로(또는 모닥불) 설치. 첫 요리는 재료가 흔한 **소고기 스튜**·**닭고기 수프**(편안함 — 자동 회복).
3. **작물 농장**: 야생 작물 채집으로 토마토·양배추·양파·쌀 씨앗 확보. 쌀은 얕은 물에, 토마토는 밧줄을 걸어 재배.
4. **전투용 상비식**: **배부름(Nourishment) 5분 접시 음식**(구운 닭·꿀 바른 햄·잡채 등)을 챙기면 전투 내내 포만 유지 → 요리사 "배고프지 않을 때 피해"와 시너지.
5. **회복용**: **편안함(Comfort) 5분 그릇 음식**으로 자동 회복. 해로운 효과는 **핫 코코아**로 제거.
6. **여유 되면**: 술통으로 김치·육포·치즈 발효, 발효주로 **Raging(공속)** 같은 전투 버프 활용.
