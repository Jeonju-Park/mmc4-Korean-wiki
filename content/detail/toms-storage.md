# 📦 Tom's Simple Storage — 공용 창고 + 집집마다 연결 상자 구축 가이드

> 거대한 중앙 공용 창고를 만들고, 각 집에 둔 상자를 그 창고 네트워크에 연결하는 방법입니다.
> **유선(케이블)** 과 **무선(비콘 링크 / 무선 터미널)** 두 방식으로 나눠 설명합니다.
> 모드 실제 데이터(레시피·툴팁) 기반.

---

## 0. 핵심 개념 먼저

Tom's Storage의 "네트워크"는 이렇게 구성됩니다:
- **인벤토리 커넥터(Inventory Connector)** = 네트워크의 **두뇌**. 네트워크당 **딱 1개**.
- 커넥터에 **맞닿은 모든 인벤토리(상자 등)** 가 하나로 합쳐집니다. 틈은 **인벤토리 트림(Trim)** 으로 이을 수 있습니다.
- **저장 터미널 / 제작 터미널** = 합쳐진 아이템을 검색·입출고하는 **접근 창구**. 인벤토리(또는 커넥터) 위에 설치.
- 멀리 떨어진 상자는 **인벤토리 케이블 + 케이블 커넥터**(유선) 또는 **비콘 인벤토리 링크**(무선)로 끌어옵니다.

> ⚠️ 한 커넥터에는 **슬롯 한도**가 있습니다(터미널에 "남은/전체 슬롯" 표시). 거대 창고는 상자가 많아 한도에 닿을 수 있으니, 필요시 **네트워크를 분리**하거나 상자를 키우세요(통/배럴은 슬롯 1칸 취급 아님 주의).

---

## 1. 공통 — 중앙 창고 본체 (유선·무선 공통)

먼저 어느 방식이든 **중앙 창고 네트워크**를 만듭니다.

### 준비물
- **나무 상자(또는 배럴) 여러 개** — 보관 용량
- **인벤토리 커넥터 ×1**
- **인벤토리 트림 ×여러 개** (상자 사이 틈 잇기, 선택)
- **저장 터미널 ×1** (+ 원하면 **제작 터미널 ×1**)
- 재료: 다이아몬드·엔더진주·비교기·발광석·유리·나무 다수

### 구축
1. 벽면에 **상자를 잔뜩** 배치(서로 붙이거나, 한 덩어리가 되도록).
2. 그 상자 더미에 **인벤토리 커넥터**를 1개 맞닿게 설치 → 닿은 상자들이 모두 하나로 합쳐집니다.
3. 떨어진 상자는 **인벤토리 트림**으로 이어 붙입니다(트림이 빈칸을 메워 연결).
4. 아무 상자/커넥터 면에 **저장 터미널**(또는 **제작 터미널**)을 설치 → 우클릭하면 전체 재고를 검색·입출고.
5. 터미널에서 **정렬(이름/수량/모드)·검색·우클릭 모드(AE식/RS식/바닐라식)** 를 톱니로 설정.

> 💡 **제작 터미널**은 내장 제작대가 있어 네트워크 재고로 바로 제작합니다. 창고엔 제작 터미널을 추천.

---

## 2. 🔌 유선(케이블) 방식 — 집 상자를 케이블로 창고에 연결

각 집의 상자를 **인벤토리 케이블**로 중앙 네트워크까지 물리적으로 잇습니다. 가장 안정적이고 저렴합니다.

### 추가 준비물
- **인벤토리 케이블 ×다수** (집까지의 거리만큼, 6개씩 제작)
- **인벤토리 케이블 커넥터 ×(집 수 + 1)** — 네트워크 끝 1개 + 각 집 상자마다 1개
- (선택) **인벤토리 케이블 커넥터(필터) ×집마다** — 특정 아이템만 들어오게 거르고 싶을 때
- (선택) **커스텀 케이블(Framed)** — 벽 텍스처로 위장해 숨기기

### 구축 순서
1. 중앙 네트워크의 인벤토리 커넥터 쪽에 **케이블 커넥터**를 1개 답니다(네트워크 출발점).
2. 거기서 **인벤토리 케이블**을 각 집까지 쭉 연결(벽·바닥 속으로 매설 가능, 커스텀 케이블로 위장).
3. 각 집에서 **상자 면에 케이블 커넥터**를 붙이고, 케이블을 그 커넥터에 연결.
4. 이러면 집 상자가 **중앙 네트워크의 일부**가 됩니다 → 집 상자에 넣은 물건이 창고 터미널에 나타나고, 어느 터미널에서든 접근.

### 유선 팁
- **케이블 커넥터 1개당 연결 가능한 케이블 수에 상한**이 있습니다(설정 `인벤토리 케이블 커넥터 최대 케이블 수`). 너무 길면 중간에 커넥터를 더 쓰세요.
- **양쪽 청크가 로드**되어 있어야 작동합니다(멀리 떨어진 집은 청크 로딩 주의).
- **필터 케이블 커넥터**로 "이 집 상자는 광물만" 같은 분류 입고가 가능.
- 케이블은 **커스텀(Framed) 케이블 + 페인트 키트**로 주변 블록처럼 위장해 깔끔하게.

## 3. 📡 무선 방식

무선은 두 층위가 있습니다. 목적에 따라 고르세요.

### 3-A. 무선 "접근" — 창고를 어디서나 열기 (저렴·권장)
집 상자를 따로 두지 않고, **각자 무선 터미널로 중앙 창고에 접근**하는 방식.
- **무선 터미널(Wireless Terminal)**: 들고 다니며 **일정 범위 내**에서 창고 터미널에 접근(범위는 설정값).
- **고급 무선 터미널(Advanced Wireless Terminal)**: 터미널에 **Shift+우클릭으로 바인딩** → 더 넓은 범위. **근처에 비콘**을 세우면 **같은 차원 어디서나**(또는 차원 너머까지) 접근 가능.
- 사용법: 창고 터미널을 한 번 바인딩 → 이후 집·필드 어디서든 단축키로 열어 입출고.

> 가장 간단한 "공용 창고"는 **중앙 네트워크 + 모두가 고급 무선 터미널**을 갖는 형태입니다. 집마다 상자가 꼭 필요 없다면 이게 제일 편합니다.

### 3-B. 무선 "인벤토리 링크" — 집 상자를 비콘으로 무선 연결 (고급·비쌈)
케이블 없이 **집 상자를 무선으로 창고 네트워크에 편입**시키는 방식. **비콘**이 필요합니다.

원리: **인벤토리 케이블 커넥터를 비콘 위에 설치**하면 **무선 인벤토리 링크**가 됩니다.
- 커넥터 모드 2가지:
  - **인벤토리에 대한 무선 접근을 제공(Provide Inventories for Remote Access)** — 내 상자를 무선으로 내보냄
  - **무선 인벤토리 접근(Access remote Inventories)** — 다른 곳의 상자를 무선으로 받아옴
- **채널 이름**을 정하고, **공용(Public)/개인(Private)** 채널을 선택해 같은 채널끼리 연결.

#### 구축
1. **각 집**: 상자에 케이블 커넥터를 달고 → 그 커넥터를 **비콘 위**에 설치 → 모드 **"무선 접근 제공"**, 채널명 예: `마을창고` (공용).
2. **중앙 창고**: 네트워크 커넥터에 케이블 커넥터를 달고 → **비콘 위**에 설치 → 모드 **"무선 인벤토리 접근"**, 같은 채널 `마을창고`.
3. 이제 집 상자들이 **무선으로 창고 네트워크에 합류** → 창고 터미널에서 전부 보입니다.

#### 무선 링크 주의
- **비콘이 필요**합니다. 같은 차원 접근은 설정된 **비콘 레벨**(피라미드 단계)이 요구되고, **차원 너머**는 더 높은 레벨이 필요합니다. (서버 설정 `inv_link_beacon_lvl` 값에 따라 **비콘 블록 1개만으로** 되게 완화돼 있을 수도 있음 — 관리자에게 확인)
- 비콘은 **철/금/다이아/에메랄드/네더라이트 블록 + 네더의 별(위더 처치)** 이 들어가 **집집마다 두려면 비쌉니다.**
- **양쪽 청크 로드** 필수.

## 4. 제작법 (주요 부품)

> `판자`=아무 판자, `상자`=나무 상자, `다이아`=다이아몬드, `엔더진주`=엔더 진주.

| 부품 | 제작대 배치 |
|---|---|
| **인벤토리 커넥터** | `판자 비교기 판자` / `상자 다이아 상자` / `판자 엔더진주 판자` |
| **저장 터미널** | `판자 비교기 판자` / `상자 발광석 유리` / `판자 비교기 판자` |
| **제작 터미널** | `제작대 다이아 제작대` / `다이아 저장터미널 다이아` / `제작대 다이아 제작대` |
| **인벤토리 케이블 (6개)** | `나무막대 판자 나무막대` / `나무뚜껑문 상자 나무뚜껑문` / `나무막대 판자 나무막대` |
| **인벤토리 케이블 커넥터** | `(빈) 석영 판자` / `케이블 상자 엔더진주` / `(빈) 다이아 판자` |
| **무선 터미널** | `판자 비교기 판자` / `망원경 발광석 유리` / `판자 엔더진주 판자` |
| **고급 무선 터미널** | 대장장이 작업대: 네더라이트 주형 + 무선 터미널 + 네더라이트 주괴 |

- **인벤토리 트림**·**필터 케이블 커넥터**·**레벨 측정기**·**인벤토리 호퍼**·**프록시** 등 보조 부품도 있습니다(JEI에서 레시피 확인).

## 5. 운영 팁

- **슬롯 한도 관리**: 터미널 상단/툴팁에 "남은/전체 슬롯". 거대 창고는 상자가 많아 한도에 닿으면 일부가 안 보일 수 있음 → 상자를 **배럴/대형 보관**으로 바꾸거나 네트워크 분리.
- **레벨 측정기(Level Emitter)**: 특정 아이템이 "N개 미만/초과"일 때 레드스톤 신호 → 자동 농장 on/off 연동.
- **인벤토리 호퍼**: 네트워크에 자동 입고/출고(필터 가능) — 농장 산출물 자동 보관에.
- **중복 상자 방지**: 같은 인벤토리를 케이블과 인접 양쪽으로 잇지 마세요("루프 감지/중복" 경고).
- **터미널 조작 모드**: 톱니로 **AE식/RS식/바닐라식** 클릭, **정렬(수량/이름/모드)**, **검색 동기화**, **제작 시 플레이어 인벤토리에서 끌어오기** 등을 취향대로.
- **위장**: 커스텀(Framed) 케이블·트림 + **페인트 키트**로 주변 블록처럼 칠해 깔끔하게.
- **설정 `인벤토리 연결에 트림만 허용`** 이 켜진 서버라면, 상자끼리 그냥 붙여도 자동 연결이 안 되고 **반드시 트림**으로 이어야 하니 주의.

## 6. 추천 구성 (집집마다 연결 상자)

| 목표 | 추천 방식 |
|---|---|
| 저렴·안정적, 집이 가까움 | **유선**: 중앙 네트워크 → 케이블로 각 집 상자 연결 + 다들 **무선 터미널**로 접근 |
| 집이 멀거나 케이블 매설 싫음, 자원 여유 | **무선 링크**: 각 집 상자를 **비콘 케이블 커넥터(채널 공유)** 로 무선 편입 |
| 그냥 공용 창고만 같이 쓰면 됨 | **무선 접근만**: 중앙 창고 1개 + 전원 **고급 무선 터미널**(비콘으로 무한 범위) |

## 7. 구축 체크리스트
- [ ] 중앙 창고: 상자 더미 + **인벤토리 커넥터 1개** + (트림) + **제작/저장 터미널**
- [ ] (유선) 각 집 상자에 **케이블 커넥터** + 중앙까지 **케이블** 연결
- [ ] (무선 링크) 각 집·창고에 **비콘 + 케이블 커넥터**, 같은 **채널명**·공용 설정
- [ ] (무선 접근) 플레이어마다 **무선/고급 무선 터미널** 바인딩
- [ ] 슬롯 한도·청크 로드·중복 연결 점검
