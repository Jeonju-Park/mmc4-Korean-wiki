# 🍳 요리사(Cook) 직업 플레이 가이드

> Medieval MC MMC4 — **Skill Tree(패시브 스킬트리)** 모드의 6개 직업 중 하나
> 요리사 = **음식을 먹고 배를 채워 강해지는 근접 탱커/브루저**. 직접 만든 음식이 회복·버프·피해를 강화한다.
> 이 가이드는 모드팩 실제 데이터(`skilltree`)에 근거합니다.

---

## 1. 먼저 — Skill Tree(직업) 시스템이란?

- 종족(Origins)과 **별개**인 **패시브 스킬트리**입니다. 거대한 노드 맵에서 능력치를 찍습니다.
- **스킬 포인트 얻는 법**: **경험치 레벨을 소모**해 스킬 포인트를 구매(`Buy Skill Point`). 즉 레벨을 올려 포인트를 모읍니다.
- **직업 선택**: 스킬트리에서 원하는 **직업 노드(클래스)**를 찍는 것이 곧 전직. 6직업 — **연금술사 / 대장장이 / 요리사 / 마법부여사 / 사냥꾼 / 광부**.
- **노드 등급**: 일반(작은 노드) → **주요(Notable)** → **키스톤(Keystone, 강력하지만 트레이드오프)**.
- **게이트웨이(Gateway)**: 직업 트리를 서로 잇는 통로. **요리사 = "영적 관문(Spiritual Gateway)"** → 같은 관문을 쓰는 **연금술사** 트리와 연결됩니다(혼합 빌드 가능).
- **초기화**: **망각의 두루마리(Amnesia Scroll)** 사용 시 스킬트리 리셋(단, **스킬 포인트 일부 손실** 경고).

---

## 2. 요리사 직업 — 핵심 콘셉트

요리사는 **"잘 먹어서 강해지는" 근접 직업**입니다. 세 가지 축으로 작동합니다:

1. **제작 음식 보너스 (Crafted Food)** — **내가 직접 만든 음식**을 먹으면 추가 효과(재생/회복/포만감/피해 보너스)를 받음.
2. **배부름 = 전투력** — **허기가 가득 찰수록(배고프지 않을 때) 피해 증가**.
3. **탱키한 근접 생존** — 막기(Blocking)·최대 체력·공격 속도·막을 때 회복(Life on Block) 노드가 풍부.

> 즉 요리 모드(농부의 기쁨 등)로 좋은 음식을 만들어 **항상 배를 채우고**, 그 상태로 **묵직하게 근접전**을 하는 빌드입니다. (요리 시스템은 `모드팩_플레이_가이드.md` 6-2 참고)

## 3. 주요 노드 정리

### 🍖 음식·제작 계열 (직업의 정체성)
| 노드 | 효과 |
|---|---|
| **제작 음식 재생 (Crafted Food Regeneration)** | 직접 만든 음식 섭취 시 재생 |
| **제작 음식 회복 (Crafted Food Healing)** | 제작 음식이 체력 회복 |
| **제작 음식 포만감 (Crafted Food Saturation)** | 제작 음식의 포만감↑ (오래 배부름) |
| **제작 음식 피해 보너스 (Crafted Food Damage Bonus)** | 제작 음식 섭취 후 피해↑ |
| **암브로시아 과일 (Ambrosia Fruit)** ⭐주요 | 제작 음식 계열 강화 주요 노드 |
| **큰 그릇 (Large Servings)** ⭐숙련 | 요리사 마스터리 |

### ⚔️ 공격 계열
| 노드 | 효과 |
|---|---|
| **배고프지 않을 때 피해 (Damage if not Hungry)** | 허기가 찼을 때 **피해 대폭 증가** (핵심) |
| **공격 속도 (Attack Speed)** | 근접 공격 속도↑ |
| **묵직한 일격 (Heavy Strike)** ⭐주요 | 공격 주요 노드 |
| **매운 음식 (Spicy Meal)** ⭐주요 | 치명타 관련 주요 노드 |
| **에너지 비축 (Energy Reserve)** ⭐주요 | 공격 속도 주요 노드 |

### 🛡️ 방어·체력·회복 계열
| 노드 | 효과 |
|---|---|
| **막기 (Blocking)** | 방패 막기 확률↑ (다수) |
| **방어 시 생명 (Life on Block)** | 막을 때 체력 회복 |
| **최대 체력 (Max Health)** | 최대 HP↑ |
| **두꺼운 팔 (Thick Arms)** ⭐주요 | 방어 주요 노드 |
| **간식 (Snack)** ⭐주요 | 회복 주요 노드 |
| **건강한 식단 (Healthy Diet)** ⭐주요 | 체력 주요 노드 |

### 🔑 키스톤 (강력 + 트레이드오프)
| 키스톤 | 테마 |
|---|---|
| **과체중 (Overweight)** | 방어 강화형 |
| **채식주의 (Vegetarianism)** | 제작 음식·방어 조건부 강화 |
| **뜨거운 음식 (Hot Food)** | 제작 음식 공격형 |
| **비만 체형 (Fat Body)** | 공격 강화형 |

> ⚠️ 키스톤은 강력한 만큼 조건/페널티가 따릅니다(예: 식단 제한, 이속·민첩 트레이드오프). 정확한 수치는 게임 내 노드 툴팁에서 확인하세요.

## 4. 하위 직업(서브클래스) — 2갈래 선택

요리사 트리 안에서 한쪽 전문화를 고릅니다.

### 🪓 광전사 (Berserker) — 공격 특화
- **제작 도끼 치명타 확률 (Crafted Axes Crit Chance)** + 공격 속도 위주.
- 주요/숙련: **망나니의 도끼(Headsman's Axe)**, **죽음의 문턱(Verge of Death)**, 마스터리 **피의 베일(Blood Veil)**.
- 콘셉트: **체력이 낮을수록 강해지는 저체력 공격형 도끼 딜러.** 위험하지만 폭발적인 DPS.

### 🎣 어부 (Fisherman) — 유틸/탱크 특화
- **낚시 경험치 (Experience from Fishing)** + 최대 체력 + 방어/막기.
- 주요/숙련: **숙련된 어부(Experienced Fisherman)**, **행운의 어부(Lucky Fisherman)**, 마스터리 **바다의 선물(Sea Gift)**.
- 콘셉트: **낚시로 경험치·전리품을 벌며 단단하게 버티는** 생활형 탱커. 식재료(생선) 수급과도 시너지.

### 특수 노드
- **유해 연구 (Studying Remains)** — 요리사 특수 노드.
- **영적 관문 (Spiritual Gateway)** — 연금술사 트리로 연결(물약·회복 보조 혼합 빌드 가능).

---

## 5. 추천 빌드 흐름

1. **요리사 클래스 노드**를 찍어 전직.
2. **제작 음식 계열(재생·회복·포만감)**을 먼저 올려 **생존력 기반**을 만든다. → 요리 모드로 좋은 음식을 상시 제작·휴대.
3. **배고프지 않을 때 피해 + 공격 속도 + 막기**를 섞어 **묵직한 근접 브루저** 완성.
4. 서브클래스 선택:
   - 화끈한 딜을 원하면 **광전사(도끼·저체력 폭딜)**,
   - 안정적 생활·탱킹을 원하면 **어부(낚시·체력·방어)**.
5. 여유 포인트로 **영적 관문**을 타 연금술사 트리의 회복/물약 노드를 보조 픽업.

---

## 6. 시너지 & 운영 팁

- 🍳 **요리 모드와 직결**: "제작 음식 보너스"는 **직접 만든 음식**에서 발동합니다. **농부의 기쁨(조리용 냄비·프라이팬)** 으로 든든한 요리를 만들어 먹는 습관이 곧 전투력입니다. 차원별 딜라이트 모드의 고급 요리도 좋습니다.
- 🍗 **항상 허기를 가득 유지**: "배고프지 않을 때 피해" 때문에 **포만 상태가 곧 딜**입니다. 전투 전 충분히 먹어두세요. 포만감(Saturation) 노드로 배부름을 오래 유지.
- 🛡️ **방패 활용**: 막기·방어 시 생명 노드가 많으니 **방패를 든 근접전**이 효율적입니다.
- 🧪 **종족과 조합**: 탱키한 종족(예: **예티** — 높은 체력)과 궁합이 좋습니다. 회복이 정상 작동하는 종족이면 제작 음식 회복도 그대로 살아납니다.
- ♻️ **리셋은 신중히**: 망각의 두루마리는 포인트를 일부 잃으니 빌드를 충분히 굳힌 뒤 사용.

## 7. 요리 방식(조리도구) — 무엇으로 만드나

이 모드팩의 요리는 **농부의 기쁨 계열 4가지 조리 방식**을 씁니다.

| 조리도구 | 용도 | 비고 |
|---|---|---|
| **조리용 냄비 (Cooking Pot)** | **스튜·수프·카레·죽 등 그릇 음식** | ⭐ **효율 음식(Comfort/Nourishment)이 전부 여기서** 나옴. 요리사 보너스 대상 |
| **도마 (Cutting Board)** | 칼로 재료 손질(다진 고기·슬라이스·반죽) | 요리 전 단계 (자세히는 별도 도마 사용법 참고) |
| **모닥불 / 훈연기** | 슬라이스 고기·생선·계란 가열 | 단순 가열, **버프 없음** |
| **술통 (Keg)** | 발효(치즈·김치·육포·술) — *Brewin' and Chewin'* | 온도·발효시간 필요 |

> ⚠️ 이 모드팩엔 별도 프라이팬 전용 레시피가 없어, 프라이팬도 조리냄비 레시피를 사용합니다.
> 💡 **요리사 직업의 "제작 음식 보너스"는 조리용 냄비로 직접 만든 그릇 음식과 정확히 맞물립니다.** 냄비 요리를 주력으로 삼으세요.

> 참고: 모드 `chefsdelight`는 음식이 아니라 **주민 직업(셰프/요리사)을 추가**하는 모드입니다. 실제 음식은 farmersdelight + 차원별 딜라이트 모드(딜라이트풀·던전스·트와일라잇·네더스·에테르·엔더스·오션스·쿼크 딜라이트)가 제공합니다.

## 8. ⭐ 효율 좋은 요리 (Comfort / Nourishment)

농부의 기쁨에는 **그릇/접시 음식에만 붙는 두 가지 강력한 효과**가 있습니다. 둘 다 **직접 만든 냄비 음식**에서만 나오므로 요리사에게 최적입니다.

| 효과 | 내용 | 붙는 음식 |
|---|---|---|
| **편안함 (Comfort)** | **배고픔과 무관하게 지속 생명력 회복** | 따뜻한 그릇 음식(스튜·수프) |
| **배부름 (Nourishment)** | **(회복 시 제외) 배고픔이 줄지 않음** = 오래 안 배고픔 | 접시 음식(구이·파스타·볶음밥) |

### 🥇 최고 효율 — 포만 14 + 효과 5분(300초)

**편안함(지속 회복) 톱티어** ← 생존/탱킹에 추천
- **닭고기 수프 · 볶음밥 · 호박 수프 · 대구 스튜 · 우동(누들 수프)** (포만 14 / Comfort 300초)

**배부름(배고픔 정지) 톱티어** ← "배고프지 않을 때 피해" 딜 유지에 최적
- **구운 닭 · 꿀 바른 햄 · 셰퍼드 파이 · 속 채운 호박 · 양고기 구이 · 잡채 · 오징어 먹물 파스타** (포만 14 / Nourishment 300초)

> 🍗 요리사는 **"배고프지 않을 때 피해"** 노드가 핵심이므로, **배부름(Nourishment) 5분 음식**을 상비하면 전투 내내 만피 포만 상태로 딜을 유지할 수 있습니다. 탱킹·생존 위주라면 **편안함(Comfort) 음식**으로 자동 회복.

### 🍯 특수 효과 음식 (상시 섭취용)
| 음식 | 효과 |
|---|---|
| **사과주 (Apple Cider)** | 이동속도 증가 60초 |
| **발광 열매 커스터드** | 점프 강화 5초 |
| 믹스/과일 샐러드 | 재생 5초 |

### 🍖 잔치 음식(블록형) — 파티 보급용
구운 닭·꿀 바른 햄·셰퍼드 파이·속 채운 호박·라이스롤 모듬은 **블록으로 설치해 여러 번 퍼먹을 수 있습니다.** 서버 파티/콜로니 보급에 최적.

## 9. 대표 요리법 (조리용 냄비 + 그릇)

> 형식: **산출물** = 재료 → 그릇. (재료는 대체 가능 항목이 많음)

**효율 음식**
1. **닭고기 수프** = 생닭 + 당근 + 채소 + 샐러드 재료 → 그릇 *(포만14/편안함 300s)*
2. **호박 수프** = 호박 조각 + 샐러드 재료 + 생돼지고기 + 우유 → 그릇 *(편안함 300s)*
3. **우동(누들 수프)** = 파스타 + 익힌 계란 + 말린 켈프 + 생돼지고기 → 그릇 *(편안함 300s)*
4. **소고기 스튜** = 생소고기 + 당근 + 감자 → 그릇 *(편안함 180s)*
5. **속 채운 호박** = 쌀 + 양파 + 갈색버섯 + 감자 + 열매 + 채소 → (그릇=호박) *(배부름 300s)*
6. **사골국** = 뼈 + (발광 열매 / 버섯 / 매달린 뿌리) → 그릇 *(편안함 60s)*

**특수 효과**
7. **사과주** = 사과 + 사과 + 설탕 → (그릇=유리병) *(이동속도 60s)*
8. **발광 열매 커스터드** = 발광 열매 + 우유 + 계란 + 설탕 → 그릇 *(점프 강화)*

**발효(술통, Brewin' and Chewin')**
9. **김치** = 양배추 + 채소 + 켈프 → 술통 발효 (김치 ×2)
10. **육포** = 생고기 ×3 → 술통 발효 (육포 ×3)
11. **쌀술** = 쌀 + 갈색버섯 (베이스: 물) → 술통 발효

> 도마로 **다진 고기·슬라이스·반죽**을 먼저 만들어두면 냄비 요리가 수월합니다. 정확한 재료는 게임 내 JEI(레시피 보기)에서 확인하세요.

---

## 10. 자주 하는 실수

- ❌ 배고픈 채로 전투 → "배고프지 않을 때 피해"가 안 들어와 딜 손실. 먹고 싸우기.
- ❌ 아무 음식이나 의존 → 보너스는 **제작 음식**에서 발동. 요리해서 먹기.
- ❌ 광전사인데 고체력 유지 → 저체력에서 강해지는 설계와 어긋남(빌드 의도 확인).
- ❌ 키스톤을 무지성으로 픽 → 페널티 확인 필수. 툴팁을 읽고 선택.
