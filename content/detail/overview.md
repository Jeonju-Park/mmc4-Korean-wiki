## 1. 장비 강화의 핵심 — Apotheosis

이 모드팩에서 **장비를 강하게 만드는 뼈대**입니다. 무기·방어구 종류가 무엇이든, 결국 이 시스템으로 강화합니다.

### 1-1. 레어도(등급) — 약 → 강
```
흔함 → 드뭄 → 희귀함 → 영웅 → 신화 → 고대 → [가보] → [아티팩트] → [난해(Esoteric)]
Common Uncommon Rare  Epic  Mythic Ancient Heirloom Artifact  Esoteric
```
- 앞 6단계는 Apotheosis 기본, 뒤 3단계(가보/아티팩트/난해)는 **Apotheotic Additions** 추가 최상위 등급.
- 등급이 높을수록 **접사(특수효과) 개수와 위력**이 강해집니다.

### 1-2. 접사(Affix) 아이템
- **"접사 아이템(Apothic Affix Item)"** = 특수효과가 붙은 장비. **던전 전리품**과 **아포틱 보스 처치**로 드랍됩니다. (보스는 "Will Drop: OO"로 드랍 아이템을 예고)
- 장비 종류별 접사 풀이 다릅니다: 검 / 중무기 / 원거리 / 방어구 / 방패 / 채굴도구.
- 접사 예시: **흡혈(Vampiric)**, 관통(Piercing), 빙결(Glacial), 지옥(Infernal), 낙뢰(Thunderstruck), 우아함(of the Duelist) 등. 방어구는 축복(of Health)·강철화 등.

### 1-3. 젬(Gem)·소켓(Socket)
- 장비에 **소켓**을 뚫고 **젬**을 박아 능력치를 더합니다.
- **젬 품질**(이름 접두어로 구분, 약→강):
  ```
  금이 간 → 갈라진 → 결점이 있는 → (수식어 없음) → 결점이 없는 → 완벽한
  Cracked  Chipped   Flawed       (epic)        Flawless     Perfect
  ```
  (Apotheotic Additions는 위로 Ancestral/Time-Lost/Cosmic 추가)
- 젬마다 장착 가능한 장비 종류가 정해져 있습니다(근접무기/원거리/방어구/곡괭이 등).
- 대표 젬: 수호자의 보석·태양/달의 보석·전쟁 군주의 보석·번개의 보석·굶주린 피의 군주의 보석 등.

### 1-4. 강화 작업대 — 실제 흐름
| 작업대 | 역할 |
|---|---|
| **대장장이 작업대 (Smithing Table)** | 장비 + 젬 → **젬 소켓 장착** |
| **재단조 작업대 (Reforging Table)** | 재탄생의 인장 + 재료로 **접사 다시 굴리기** |
| **보석 절단 작업대 (Gem Cutting Table)** | 보석 가루로 **젬 품질 업그레이드** |
| **증강 작업대 (Augmenting Table)** | 마법 부여의 인장으로 **개별 접사 강화** |
| **분해 작업대 (Salvaging Table)** | 접사 아이템 분해 → 희귀 재료 회수 |

> 🔰 **초보 강화 순서**: ① 던전/보스에서 접사 아이템 획득 → ② 마음에 안 들면 **재단조**로 접사 재굴림 → ③ **소켓의 인장**으로 소켓 추가 → ④ **보석 절단**으로 젬 품질 업 → ⑤ **대장장이 작업대**로 젬 장착.
> ⚠️ 소켓 박은 젬을 빼면(Sigil of Withdrawal) **아이템이나 젬이 파괴**되니 신중히.

## 2. 무기 티어

### 2-1. 최상위 라인 — Ice and Fire (드래곤 무기)
재료 티어 (약 → 강):
```
은(Silver) → 드래곤 뼈(Dragon Bone) → 드래곤스틸(Dragonsteel)
```
- **드래곤 뼈 검**: 화염(Flamed)/빙결(Iced)/번개(Lightning) 속성 3종.
- **드래곤스틸**(최상위): **불/얼음/번개** 3계열, 각 계열에 검·도끼·도구 + **풀 방어구**.
  - 제작: 드래곤 피 + 철 → 드래곤스틸 주괴 → 무기/방어구.

### 2-2. 보스 무기 — Cataclysm
특정 보스를 잡아야 얻는 강력한 무기들:
| 무기 | 특성 |
|---|---|
| **파이널 프랙탈 (Final Fractal)** | 대상 최대 체력의 3% 추가 피해 |
| **츠바이엔더 (Zweiender)** | 뒤에서 공격 시 200% 피해 |
| **영혼 분쇄자 (Soul Render)** | "Rush of Render" 돌진기 |
| **저주받은 활 (Cursed Bow)** | 팬텀 화살 3발 발사 |
| **수호/보루의 건틀릿** | 끌어당기기 / 밀치고 돌진 |
| **견착식 위더·공허 무기** | 우클릭 미사일/곡사포 |
| **타이달 클로 (Tidal Claws)** | 갈고리 + 촉수 피해 |

### 2-3. 모듈형 — Tetra
- **무기를 부품 조립**으로 만드는 시스템(고정 등급명 없음). 검·활·쇠뇌·방패·도구.
- **작업대 → 대장간 작업대**(대장간 망치 필요)에서 제작. **홀로스피어**로 부품/재료 적용 가능 여부 확인.
- 재료의 **경도·밀도·유연성**이 성능을 결정. 전문서(Treatise)로 가공법을 해금.

### 2-4. 무기 종류 다양화 모드
- **Dungeons and Combat**: 단검·롱소드·대검·클레이모어·창·글레이브·낫·카타나·워해머 등 + 다양한 금속 티어(Cobalt/Crimson/Ebony/드래곤스틸 등). *(현재 한글 미번역)*
- **사무라이 왕조 (Samurai Dynasty)**: 카타나·와키자시·오다치·나기나타·쿠나이 등 일본풍 무기. 철<금<다이아<네더라이트 + 보석 변형 세트. 고급 카타나(오니/키츠네 등). *(현재 한글 미번역)*
- **던전 딜라이트 (Dungeons Delight)**: 나이프·클리버(요리겸용 무기). 부싯돌→철→금→다이아→네더라이트→화염.
- **Better Combat**: 무기 자체가 아니라 **전투 방식 변경 모드**. 공격에 **범위·콤보·양손/페인트** 개념을 추가 → 모든 무기의 휘두름이 더 전술적으로 바뀝니다.

## 3. 활·원거리 무기

| 무기 | 출처 | 특징 |
|---|---|---|
| 활 / 석궁 | 바닐라 | 기본 |
| **저주받은 활** | Cataclysm | 팬텀 화살 3발 |
| **드래곤 뼈 활** | Ice and Fire | 드래곤 뼈 제작 |
| **얼음/엔더/유도/삼연활** | Twilight Forest | 보스·구조물 보상 |
| **마법사의 활/석궁 (Enchanter's Bow)** | Ars Nouveau | 주문 발사 |
| **자동 장전 석궁** | Iron's Spells | 자동 장전 |

**특수 화살**: 흑요석 화살(+피해), 광두 화살(출혈), 폭발 화살, 채굴 화살(Apotheosis) / 드래곤 뼈·히드라(독+흡수)·해류 화살(Ice and Fire) / 팬텀·공허 산탄 화살(Cataclysm) 등.

**원거리 강화**:
- **Apotheosis 원거리 접사**(유선형·엘프·민첩·관통 등) + 원거리용 젬.
- **유물(Relics) 화살통**: 화살 내부 보관 + "도약(Leap)" 능력.
- **스킬트리 사냥꾼(Hunter)**: 저격·속사·정밀(크리)·흡혈 화살 등 활 특화 노드. + 화살통 티어(뼈/장갑/다이아/화염/치유 등).

## 4. 방어구 세트와 티어

### 4-1. 최상위 세트
- **드래곤스틸 세트** (Ice and Fire): 불/얼음/번개 계열 풀 방어구 — 사실상 최종 티어.
- **드래곤 비늘 갑옷 (Dragonscale)**: 드래곤 브레스 방어력 증가.

### 4-2. Cataclysm 보스 세트 (특수효과형)
| 세트 | 핵심 효과 |
|---|---|
| **이그니티움 (Ignitium)** | 화염 부여·엘리트라 결합·용암 보행. **네더라이트 장비에 업그레이드** |
| **커스뮴 (Cursium)** | 벽 너머 발광·**사망 시 부활(체력5+무적)**·회피·낙하 감소. **네더라이트 업그레이드** |
| **본 렙타일 (Bone Reptile)** | 초기 보스 세트 |

### 4-3. Ice and Fire 몹 세트
- **데스웜 키틴**(3색), **미르멕스 키틴**(사막/정글), **해류 수호자(Tide Guardian)**(수중 호흡·물에서 강화), 히포그리프 탈것 갑옷(철<금<다이아).

### 4-4. 테마 세트
- **사무라이 갑옷**(철<금<다이아<네더라이트 + 보석 변형), 마법 계열 로브(각 마법 모드별), 이단자 세트(Tome of Blood) 등.

> 어떤 세트든 **Apotheosis 젬·소켓·접사**로 추가 강화하는 것이 정석입니다.

## 5. 마법 체계 총정리

이 모드팩은 **마법 모드가 매우 많습니다.** 각자 콘셉트가 다르니 취향에 맞게 1~2개를 골라 키우세요.

| 모드 | 테마 | 자원 | 한 줄 요약 |
|---|---|---|---|
| **Iron's Spells 'n Spellbooks** | 정통 주문서 마법사 | 마나 | **모드팩 핵심.** 학파별 주문을 두루마리→주문서에 각인해 시전 |
| **Ars Nouveau** | 글리프 자작 주문 | 소스 | 형태+효과+증강 글리프를 엮어 **나만의 주문** 제작 |
| **Eidolon** | 흑마법/네크로맨시 | 마나(기도) | 기호 조합 챈트 + 의식. **언데드/흡혈 특화** |
| **Blood Magic** | 피·희생 | LP·악마의 의지 | 피의 제단에 생명을 바쳐 인장·의식 운용 |
| **Forbidden Arcanus** | 금단의 연금 | 4대 정수 | **헤파이스토스 화로**로 강력 아이템 의식 제작 |
| **Tome of Blood** | Ars + Blood 융합 | 악마의 의지 | 의지를 소비하는 다크 캐스터 글리프 |
| **Wizards Reborn** | 수정·연금·증기 | 위센 | 원소 수정을 렌즈로 시전 + 연금/자동화 |
| **Ice and Fire: Spellbooks** | 드래곤 주문 | 마나 | Iron's Spells의 드래곤(화염/얼음/번개) 확장 |

### 5-1. Iron's Spells — 시작 가이드 (가장 중요)
1. 약한 주문서(Flimsy Journal)로 입문.
2. **각인 테이블**(두루마리→주문서 각인) + **두루마리 대장간**(잉크+재료로 두루마리 제작) 설치.
3. **잉크 등급**(일반→고급→희귀→영웅→전설)이 만들 수 있는 주문 희귀도를 제한.
4. **주문서 티어**: 약한 일기 → 철제 책 → 수습생의 주문서 → 마법이 부여된 주문서 → **고대의 서**.
5. **학파 10종**: 화염·얼음·번개·신성·엔더·**피**·소환(Evocation)·공허·자연·**엘드리치**. **주문력은 학파별로 따로** 올립니다.
6. **강화**: 강화 구슬(Upgrade Orb) + 룬으로 속성별 강화 구슬 제작 → **신비한 모루**로 장비/주문서에 적용.
- 직업 콘셉트: 파이로맨서(화염)·크라이오맨서(얼음)·네크로맨서(피/엘드리치)·소환사(Evocation/Ender) 등.

### 5-2. Ars Nouveau — 시작 가이드
1. **낡은 공책**(가이드북)으로 입문.
2. **필기 작업대**에서 마도서를 들고 글리프 해금·주문 제작.
3. **주입챔버**로 소스(마력) 생산·저장(마력단지).
4. **마도서 티어**: 견습생 → 마법사 → 대마법사.
5. 주문 = **형태(투사체/접촉/셀프/벽 등) + 효과(피해/치유/소환 등) + 증강(증폭/연장/관통/분열)**.

### 5-3. Eidolon — 시작 가이드 (네크로맨서 추천)
1. **지식의 책(Codex)** → 납·은·백랍 제련.
2. **밀짚 인형 + 목재 제단**으로 기도 → 마녀/사제에게 기호 학습.
3. **제단(목재→석재)**과 **인형**을 강화해 더 강한 의식 해금.
4. 자원 = 마나(제단에서 기도/제물로 충전). **기호(Sign)를 순서대로 챈트**해 주문 발동.
5. 흑마법·영혼 수확·언데드 사역·흡혈 무기가 특기. (레버넌트 종족과 시너지 ↑)

### 5-4. 그 외
- **Blood Magic**: 피의 제단 + 피의 구슬로 시작. **인장(Sigil)**·**의식(Ritual)**·**악마의 의지**(자아 무기) 3축. 진입장벽은 높지만 깊이가 큼.
- **Forbidden Arcanus**: 주문 시전보다 **재료·강화·의식 제작** 중심. 헤파이스토스 화로로 후반 강력 장비 제작.
- **Wizards Reborn**: 위센→수정 가공→비전 지팡이로 원소 주문 + 증기/연금 자동화.

## 6. 생산성·진행 모드

> ⚠️ 이 모드팩에는 **Create·Mekanism·AE2 같은 대형 테크 자동화 모드가 없습니다.** 생산/진행은 아래가 담당합니다.

### 6-1. MineColonies (마인콜로니) — 최대 규모 생산 시스템
**NPC 시민을 고용해 마을(콜로니)을 운영**합니다. 직접 짓는 게 아니라 일꾼이 채집·가공·건설합니다.
1. **보급 캠프** 또는 **보급선**을 제작해 평지/물가에 설치(1회만 가능, 최소 16×17 평지).
2. 보급품에서 **마을회관**과 **건축 도구**를 얻어 설치 → 콜로니 생성.
3. **건축사무소**를 짓고 **건축가(Builder)**를 배정 → 이후 모든 건물 자동 건설.
4. 일꾼 건물(광산·벌목장·농장·대장간·제련소·제빵소 등)을 배치해 자원 자동 생산.
5. **대학교**에서 **연구 트리**(레벨·시간 소요)로 기술 연구. **여관**에서 신규 시민 모집, **학교/병원**으로 교육·치료.

### 6-2. 요리·음식
- **농부의 기쁨 (Farmer's Delight)**: 난로 + **조리용 냄비** + 도마 + **프라이팬**(휴대용)으로 요리. 식료품 선반에 보관.
- **차원별 확장 음식 모드**: 셰프/엔더/던전/바다/네더/황혼/에테르/쿼크 딜라이트 — 각 테마 재료·요리 추가.
- **Brewin' and Chewin'**: 통(Keg)에서 **발효 양조**(맥주·미드·럼 등). 취함/만취/격노 등 음주 효과. *(현재 한글 미번역)*

### 6-3. 이동·보관·유틸
| 모드 | 역할 |
|---|---|
| **여행자 가방 (Traveler's Backpack)** | 대용량 가방. 티어: **가죽→철→금→다이아→네더라이트**. 제작·제련·픽업·자석 등 기능 업그레이드 장착 (개인용 휴대 자동화) |
| **이동석 (Waystones)** | 거점 간 순간이동. 이동석/공명석(같은 색끼리)/전송판 + 전송·귀환 주문서, 휴대용 워프 스톤 |
| **톰의 심플 스토리지 (Tom's Simple Storage)** | 케이블 없이 상자들을 묶어 **검색형 보관/제작 터미널**로 접근. 무선 터미널 지원 |
| **유물 (Relics)** | 능력을 가진 장신구. **사용 → 경험치 → 레벨업 → 능력 강화**, 품질(1~5) 시스템 |
| **서플멘터리 (Supplementaries)** | 장식 + 실용 잡화(유리병·자루·금고·슬링샷 등) |

## 7. 초보 추천 진행 순서

1. **정착 & 생존**: 식량·기본 장비 확보. (종족이 농사 불가면 사냥/낚시 위주)
2. **MineColonies 시작**: 보급 캠프 → 마을회관 → 건축가 → 핵심 일꾼(광산/벌목장/농장/대장간)으로 **자원 자동화 기반** 마련.
3. **보관 정리**: 톰의 심플 스토리지로 창고 통합. 여행자 가방으로 휴대 편의 확보.
4. **첫 장비**: 던전 탐험으로 **접사 아이템** 수급 → Apotheosis 작업대로 강화 시작(소켓·젬·재단조).
5. **마법 1개 선택**: 취향대로 Iron's Spells(정통)·Ars Nouveau(자작)·Eidolon(흑마법) 중 하나를 키움.
6. **보스 도전**: Cataclysm 보스 → 보스 무기/세트(이그니티움·커스뮴) 획득 → 네더라이트 장비 업그레이드.
7. **최종 장비**: Ice and Fire **드래곤스틸** 무기/방어구 + Apotheosis 풀강(완벽한 젬·고등급 접사) + 마법 주문력 극대화.
