# 🏗️ 건축 담당자용 — Structurize 건축 도구 완전 가이드

> Medieval MC MMC4 — **Structurize**(건축/설계 시스템). MineColonies의 건물 배치도 이 모드가 담당합니다.
> 서버 건축 담당자가 **건물 배치·정밀 위치 조정·자작 건물 스캔·도형 생성**까지 다루도록 상세히 정리했습니다.
> 모드 실제 데이터에 근거합니다.

---

## 0. 가장 중요한 전제 — 생존 vs 크리에이티브

⚠️ **Structurize 건축 도구의 "즉시 배치(붙여넣기)"는 크리에이티브 전용입니다.**
모드 안내문(원문): *"Structurize는 생존 모드에서 건축 도구 사용을 지원하지 않습니다. 크리에이티브로 전환하거나 MineColonies를 설치하고 MineColonies 건축가를 사용하세요."*

| 상황 | 건축 방식 |
|---|---|
| **생존 모드 (MineColonies 콜로니)** | 건축 도구로 **위치만 잡고 → 건축가(Builder)가 자원을 모아 건설**. (콜로니 운영의 정식 방식) |
| **크리에이티브 모드 (자유 건축/디자인)** | 건축 도구로 **구조물을 즉시 배치**, 스캔 도구로 **자작 건물 저장**, 도형 도구로 지형/뼈대 생성 등 모든 기능 사용 |

> 서버 건축 담당이라면 보통 **크리에이티브(또는 빌드 월드)** 에서 디자인하고 스캔해 두었다가, 생존 콜로니에서 건축가로 짓는 흐름을 씁니다.

---

## 1. Structurize 5대 도구

| 도구 | 용도 |
|---|---|
| **건축 도구 (Build Tool)** | 구조물/청사진을 미리보기로 띄워 **위치 조정 후 배치** |
| **스캔 도구 (Scan Tool)** | 영역을 지정해 **내 건물을 설계도(스키매틱)로 저장** |
| **도형 도구 (Shape Tool)** | 구·정육면체·원기둥 등 **기하 도형을 블록으로 생성** (지형/뼈대용) |
| **태그 도구 (Tag Tool)** | 자작 설계도에 **태그**(바닥 기준점 등)를 부여 |
| **캘리퍼스 (Calipers)** | 두 블록 사이 **거리/크기 측정**(선·면·정육면체) |

> 도구를 손에 들고 **블록에 우클릭**하면 해당 도구의 GUI가 열립니다. 도구가 안 보이면 크리에이티브 인벤토리에서 "Structurize" 탭을 찾으세요.

## 2. 건축 도구(Build Tool) — 핵심 사용법

### 2-1. 열기 & 구조물 선택
1. 건축 도구를 들고 **고체 블록에 우클릭** → 건축 GUI + 미리보기(반투명 구조물)가 뜹니다.
   - 안내: *"먼저 고체 블록에 도구를 클릭해 시작 위치를 선택하세요."*
2. **팩 전환(Switch Pack)** 버튼으로 **구조물 팩**을 고릅니다. (MineColonies 스타일 팩, 기본 팩 등. 제작자 표시됨)
   - 팩이 안 보이면: "발견된 구조물 팩이 없습니다" → 팩 설치 필요. 서버에선 클라 전용 팩이 비활성될 수 있음(서버 설정 필요).
3. 카테고리를 들어가 **청사진(blueprint)** 을 선택하면 미리보기가 그 건물로 바뀝니다.

### 2-2. 위치/방향 조정 (가장 중요)
미리보기를 원하는 자리에 정확히 놓는 단계입니다.
- **GUI의 화살표 버튼** 또는 **설정된 단축키**로 미리보기를 **앞/뒤/좌/우/위/아래 이동**.
- **회전(rotate)** 과 **거울 반전(mirror)** 으로 방향을 맞춤.
- **ESC** 를 누르면 GUI에서 나가 **미리보기를 직접 둘러볼 수 있고**, 다시 **건축 도구를 고체 블록에 우클릭**하면 위치를 새로 잡고 GUI로 재진입합니다.
  - 안내: *"오른쪽 화살표 버튼이나 설정된 키로 미리보기를 이동하세요. esc로 나가서 위치를 다시 잡은 뒤 우클릭으로 다시 들어올 수 있습니다."*
- 키 바인딩 **"구조물 배치(Place Structure)"** 를 설정해두면 빠르게 배치할 수 있습니다.

### 2-3. 배치 실행
화면 하단/측면의 배치 버튼:
| 버튼 | 동작 | 사용 환경 |
|---|---|---|
| **건축가에게 할당 (Assign to Builder)** | 콜로니 건축가가 자원 모아 건설 | **생존(MineColonies)** |
| **설계도 붙여넣기 (Schematic Paste)** | **즉시 배치**, 자리표시 블록 유지 | 크리에이티브 |
| **건축됨 (Constructed)** | **즉시 배치**, "건축가가 지은 것처럼" 깔끔히 | 크리에이티브 |
| **취소 (Cancel)** | 배치 취소 | 공통 |

- **설계도 붙여넣기**: *"자리표시 블록을 유지한 채 구조물을 세계에 즉시 배치합니다."*
- **건축됨(Pretty)**: *"건축가가 지은 것처럼 구조물을 세계에 즉시 배치합니다."* — 보통 이걸 추천.

### 2-4. 자리표시 블록(Placeholder)
- **자리표시 블록 표시(Show Placeholder Blocks)**: *"이 위치의 블록은 배치 시 변경되지 않습니다."* — 기존 지형을 보존할 자리.
- 설정 `자리표시를 대상 블록으로 렌더링`을 켜면 자리표시가 실제 들어갈 블록 형태로 미리 보입니다(싱글/LAN 호스트에서 정확).

### 2-5. 배치 순서(이터레이터) — 연출/효율
건물이 지어지는 순서를 고를 수 있습니다:
- **한 줄씩 (기본)** · **바깥쪽부터 원형** · **원형(바깥 줄 +1~+4 높게)** · **수학적 근접도(힐베르트)** · **무작위**.
> 콜로니 건축가의 시공 연출이나, 큰 구조물의 배치 패턴을 정할 때 사용.

### 2-6. 실행 취소 / 다시 실행
- **실행 취소/다시 실행 창(Un-Redo Window)**: 최근 작업 목록에서 **왼쪽 화살표=실행 취소, 오른쪽=다시 실행**. 잘못 배치한 구조물을 되돌릴 수 있습니다.

## 3. 스캔 도구(Scan Tool) — 내 건물을 설계도로 저장

자작 건물을 **재사용 가능한 설계도(스키매틱)** 로 만드는 핵심 도구입니다. (건축 담당의 필수 기능)

### 사용 순서
1. 스캔 도구를 들고 **첫 번째 모서리 블록을 우클릭** → "첫 번째 지점 저장됨".
2. **반대쪽 모서리 블록을 우클릭** → "두 번째 지점 저장됨". → 두 점이 만드는 **직육면체 영역**이 스캔 범위.
3. 스캔 도구를 우클릭해 GUI를 열고:
   - **스캔 이름(Scan Name)** 입력(공백 없이).
   - **자원 표시(Show Resources)** — 이 건물에 들어간 블록/수량 목록 확인. (컨테이너 내용물은 안 보일 수 있음)
   - **블록 교체(Replace)**: "변경 전(From) → 변경 후(To)" 로 특정 블록 일괄 교체. **필터된 항목 제거(Remove Filtered)** 가능.
   - **보이는/숨겨진 블록**(고체에 둘러싸인) 표시 토글, 항목 클릭 시 월드에서 강조.
4. 저장하면 설계도로 보관됩니다("스캔을 OO로 성공적으로 저장했습니다").

### 기준점(Anchor)
- 설계도의 **기준점**은 배치 시 정렬 기준입니다. 기준점이 영역 밖이면 경고 → **Shift + 우클릭**으로 영역 안의 블록에 기준점을 옮기세요.
- 기준점 후보가 여럿이면 하나를 선택하라는 메시지가 나옵니다.

### 자리표시 채우기(Fill) — 지형 융화
- **채우기(Fill)** 기능으로 설계도 윗부분에 자리표시를 자동 생성해, **주변 지형에 자연스럽게 어우러지게** 합니다.
- 파라미터: **반지름(Radius)**, **Y축 늘이기(Y-Stretch)**, **높이 오프셋(Heightoffset)**, **블록 거리(Blockdist, 기존 블록과 떨어질 최소 거리)**.

> ⚠️ 설계도에는 **크기 제한**이 있습니다("설계도가 너무 큽니다. 최대 허용 부피는 N 블록"). 너무 큰 건물은 나눠 스캔하세요.

## 4. 도형 도구(Shape Tool) — 지형/뼈대 빠른 생성

구·정육면체·원기둥 등 기하 도형을 한 번에 블록으로 채웁니다. (지형 정리, 돔/탑 뼈대, 벽 등에 유용)

- **주 블록 선택(Pick Main Block)** / **채움 블록 선택(Pick Fill Block)** — 외형/내부 블록 지정.
- **꽉 찬(Solid)** / **속 빈(Hollow)** 토글 — 속을 채울지 비울지.
- **블록 교체(Replace Blocks)** / **블록 무시(Ignore Blocks)** — 기존 블록 처리 방식.
- 크기/형태를 조절해 미리보기를 보고 배치.

## 5. 태그 도구(Tag Tool) — 설계도에 태그 부여 (고급)

자작 설계도가 **건축 도구에서 올바로 작동**하도록 블록에 의미를 부여합니다. (크리에이티브 전용)

1. **Shift + 우클릭**으로 **유효한 기준점 블록**(예: Tag Anchor Block)을 선택 → "이제 블록에 태그를 적용할 수 있습니다!"
2. **적용할 태그(Tag to apply)** 를 입력하고 블록에 **태그 적용(Apply Tags)**.
3. **태그 적용/변경 취소/초기화** 가능.

자주 쓰는 태그:
- **groundlevel** — 건축 도구가 청사진을 **바닥에 맞춰 배치**할 기준(바닥면 지정).
- **invisible** — 통합 건물·광산 갱도처럼 건축 도구 미리보기에서 **청사진 숨김**.

> MineColonies 커스텀 건물(일꾼 배정용)을 만들 때 특히 중요합니다.

---

## 6. 캘리퍼스(Calipers) — 거리/치수 측정

- 두 블록을 차례로 클릭하면 거리를 알려줍니다:
  - 같은 축: **"N 블록 길이의 선"**
  - 평면: **"N x N 블록 크기의 사각형"**
  - 입체: **"N x N x N 블록 크기의 정육면체"**
- 건물 규격을 재거나 스캔 범위를 미리 계산할 때 유용.

## 7. 서버 운영 시 주의 (건축 담당 필독)

- **구조물 팩 동기화**: 청사진은 **서버에도 같은 팩이 있어야** 배치됩니다. "요청한 청사진이 서버 측에 없습니다" 오류 = 서버 팩이 구버전이거나 클라 팩이 다름. 자작 팩은 서버에 함께 올려야 함.
- **플레이어 설계도 허용**: 서버 설정 `플레이어 설계도 허용(allowplayerschematics)` 이 켜져 있어야 플레이어가 만든 설계도를 쓸 수 있습니다. 클라 전용 팩은 서버에서 해당 옵션을 켜야 활성화됩니다.
- **권한**: 명령어 스캔/붙여넣기는 권한이 필요할 수 있습니다("명령어로 스캔할 권한이 없습니다. 대신 스캔 도구를 사용하세요").
- **설계도 캐시/크기**: 서버 `최대 캐시 설계도 수`, 설계도 **최대 바이트/부피 제한**이 있습니다. 대형 건물은 분할.
- **순간이동(빌드로 이동)**: 명령 블록을 복사/붙여넣어 텔레포트 위치를 설정할 수 있고, **건축물과 같은 차원**에 있어야 합니다. 착지 방향/거리는 설정으로 조정.

## 8. 추천 워크플로 (서버 건축 담당)

### A. 콜로니 건물 배치 (생존)
1. 건축 도구 우클릭 → 팩/청사진 선택 → 위치·회전 조정 → **건축가에게 할당**.
2. 건축가에게 자원 공급 → 자동 건설. (자세한 콜로니 운영은 마인콜로니 안내 참고)

### B. 자작 건물 제작·배포 (크리에이티브/빌드 월드)
1. 빌드 월드에서 건물을 짓는다.
2. **스캔 도구**로 두 모서리 지정 → 이름 지정 → (필요 시 블록 교체·기준점 조정·Fill) → **저장**.
3. 필요하면 **태그 도구**로 groundlevel 등 태그 부여.
4. 건축 도구에서 내 팩을 골라 **건축됨(Constructed)** 으로 즉시 배치하거나, 생존 콜로니에서 건축가로 건설.
5. **서버에 같은 구조물 팩을 올려** 동기화.

### C. 지형/뼈대 작업
- **도형 도구**로 돔·원기둥·벽 뼈대를 빠르게 깔고, 디테일을 손으로 마감.
- **캘리퍼스**로 치수 측정, **실행 취소/다시 실행**으로 시행착오 관리.

## 9. 자주 막히는 부분 (FAQ)

- **Q. 생존인데 건축 도구로 즉시 배치가 안 돼요** → 정상입니다. 즉시 배치는 크리에이티브 전용. 생존은 **MineColonies 건축가**로 지으세요.
- **Q. 팩 목록이 비어 있어요** → 구조물 팩 미설치. 서버에선 클라 전용 팩이 비활성될 수 있으니 서버 설정 확인.
- **Q. 스캔 시 "기준점이 영역 밖"** → **Shift + 우클릭**으로 스캔 영역 안의 블록에 기준점을 옮기세요.
- **Q. 설계도가 너무 크다고 떠요** → 부피 제한 초과. 건물을 분할 스캔하거나 서버 설정 상향.
- **Q. 배치한 게 잘못됐어요** → **실행 취소/다시 실행 창**에서 되돌리기.
- **Q. 미리보기를 더 잘 보고 싶어요** → **ESC**로 GUI를 나가 직접 둘러보고, 도구를 블록에 우클릭해 재진입.
