# 🎒 생산성 아이템 위키 (Medieval MC MMC4)

> **다루는 범위**: 손에 들고 쓰는 **소비형·장신구형 "아이템"** 중 이동·수집·효율·편의를 높여 주는 것들.
> 유물(Relics) 편의형, 이동 주문서, 효율 음식·물약, 채집 보조(미끼·탐지봉), 인챈트·경험치 절약 소모품을 정리합니다.
>
> **7번 위키와의 차이** → `7_생산성툴_위키.md` 는 보관·이동·가공을 책임지는 **툴 / 블록 / 장비**(배낭, 전송석 블록, 제재소, 거래소 등)를 다룹니다.
> 이 10번은 그 위에서 **들고 발동하는 아이템**을 다룹니다. 겹치는 항목(전송석·휴대용 전송석)은 여기서 간단히 정리하고 자세한 설치·제작은 7번을 참고하세요.
>
> 모든 효과/명칭은 모드 jar의 실제 언어 파일(`relics`, `waystones`, `farmersdelight`, `tide`, `georenouveau`, `apotheosis`)에서 추출했습니다.
> 일부 미번역 영문 아이템은 `영문 (의역)` 형식으로 표기합니다.

## 1. 개요 — 7번 "툴"과 무엇이 다른가

| 구분 | 7번 (생산성 툴) | 10번 (생산성 아이템) |
|---|---|---|
| 형태 | 블록 / 설치물 / 장비 | 들고 쓰는 소비품 · 장신구(유물) |
| 예시 | 전송석 블록, 배낭, 제재소, 거래소 | 매직 미러, 전송 주문서, 효율 음식, 미끼, 탐지봉 |
| 사용 방식 | 설치 후 상호작용 | 손/장신구 슬롯에서 즉시 발동 |
| 핵심 가치 | 거점·자원 인프라 | **현장에서 즉시 쓰는 시간·효율 절약** |

생산성 아이템은 5계열로 나뉩니다.

| 계열 | 대표 아이템 | 해결하는 문제 |
|---|---|---|
| 🌀 이동·귀환 | 매직 미러, 전송/귀환 주문서, 휴대용 전송석 | 거점 왕복·죽음 후 복귀 |
| 🔍 수집·채집 | 미끼, GeOre 탐지봉, 행운 낚싯줄 | 광맥·전리품·물고기 탐색 시간 |
| 🍖 유지·회복 | 무한한 햄, 편안함/배부름 음식, 사과주 | 식량 보급·자연 회복 끊김 |
| 💍 편의 유물 | 말 피리, 아쿠아/마그마 워커, 포자 주머니 | 이동 장애물·반복 작업 |
| ⏳ 시간·경험 절약 | 폐기의 책, 프리즈마틱 거미줄, 유물 경험치 병 | 인챈트·경험치 낭비 |

## 2. 이동·귀환 아이템

거점 간 왕복과 사망 후 복귀 시간을 없애는 핵심 계열. (전송석 **블록** 설치/제작 = 7번 위키 참고)

### 2-1. 전송석(Waystones) 계열 휴대 아이템

| 아이템 | 효과 | 비고 |
|---|---|---|
| **휴대용 전송석** (`warp_stone`) | 어디서나 활성화된 전송석으로 순간이동. 사용 후 재충전 필요 | "완전히 충전되지 않았습니다" 메시지 = 쿨다운 중 |
| **전송 주문서** (`warp_scroll`) | 1회용. 활성화된 전송석 중 선택해 이동 | 소모품 |
| **전송 주문서 (연결)** (`warp_scroll_bound`) | 특정 전송석에 미리 연결된 무한 사용 전송 주문서 | 연결 대상 고정 |
| **귀환 주문서** (`return_scroll`) | 가장 가까운(또는 지정) 전송석으로 귀환 | 긴급 탈출용 |
| **연결 주문서** (`bound_scroll`) | 전송석을 우클릭해 목적지를 "연결"시켜 두는 주문서 | 연결 후 반복 전송 |
| **전송 가루** (`warp_dust`) | 위 주문서·전송판 제작 재료 | 가공 소재 |
| **익숙한 파편** (`attuned_shard`) | 전송판(`warp_plate`)끼리 연결하는 파편 | 자동 전송 라인용 |

> ⚠️ 전송에는 **경험치(XP)가 소모**됩니다. 거리(블록당 XP 레벨)·차원 이동 여부에 따라 비용이 달라지며, 차원 전송은 추가 비용이 붙습니다. 끈으로 묶은 몹과 함께 이동하면 끈당 XP가 더 듭니다.

### 2-2. 유물(Relics) 이동·귀환형

| 유물 | 능력 | 효과 요약 | 획득 |
|---|---|---|---|
| **매직 미러** (`magic_mirror`) | 웜홀 | 리스폰 지점으로 순간이동. 거리·재충전(쿨타임) 있음 | 던전/전리품 상자 |
| **엔더 핸드** (`enders_hand`) | 엔드 전이 | 바라보는 대상과 **위치 교환**. 엔더맨이 중립화됨 | 전리품 상자 |
| **공간 해체기** (`space_dissector`) | 해부 | 우클릭 유지/해제로 **연결된 포탈 2개** 생성, 생명체·아이템 이동 | 전리품 상자 |
| **공간의 표식** (`spatial_sign`) | 시간 균열 | 이동 경로를 기록해 일정 시간 후 **원위치로 자동 귀환** | 전리품 상자 |
| **후렴 억제기** (`chorus_inhibitor`) | 순간이동 제어 | 후렴과 사용 시 **바라보는 지점**으로 정확히 순간이동(랜덤 X) | 전리품 상자 |

> 💡 유물은 사용할수록 경험치가 쌓여 레벨업하고 능력이 강화됩니다. 자세한 레벨/품질 시스템은 5번 무기·6번 방어구 위키의 유물 설명을 함께 참고하세요.

## 3. 수집·채집 효율 아이템

광맥·물고기·전리품 탐색에 드는 시간을 줄여 주는 보조 아이템.

### 3-1. GeOre 탐지봉 (Georenouveau) — *미번역, 의역 표기*

광물별로 짝이 있는 **탐지봉(Dowsing Rod)**. 사용 시 **행운(Magic Find)** 과 **투시(Scrying)** 를 부여해, 해당 광물(지오드)과 마법 생물이 **블록 너머로 빛나 보이게** 합니다. 또한 마법 모드의 주입실/마법 장치에 사용하면 연결된 받침대를 강조합니다.

| 탐지봉 (영문 → 의역) | 탐지 대상 |
|---|---|
| Coal GeOre Dowsing Rod (석탄 탐지봉) | 석탄 지오드 |
| Copper GeOre Dowsing Rod (구리 탐지봉) | 구리 지오드 |
| Iron GeOre Dowsing Rod (철 탐지봉) | 철 지오드 |
| Gold GeOre Dowsing Rod (금 탐지봉) | 금 지오드 |
| Lapis GeOre Dowsing Rod (청금석 탐지봉) | 청금석 지오드 |
| Redstone GeOre Dowsing Rod (레드스톤 탐지봉) | 레드스톤 지오드 |
| Diamond GeOre Dowsing Rod (다이아 탐지봉) | 다이아몬드 지오드 |
| Emerald GeOre Dowsing Rod (에메랄드 탐지봉) | 에메랄드 지오드 |
| Quartz GeOre Dowsing Rod (석영 탐지봉) | 석영 지오드 |
| Ruby / Sapphire / Topaz / Zinc Dowsing Rod (루비/사파이어/토파즈/아연 탐지봉) | 각 색 지오드 |

> 짝이 되는 **GeOre 골렘 부적(Golem Charm)** 은 해당 종류의 "싹트는 지오드(Budding Geore)" 근처에서 **각성 의식(Ritual of Awakening)** 을 수행해 얻습니다.

### 3-2. 낚시 보조 (Tide) — *미번역, 의역 표기*

> 낚시 시스템 전반(낚싯대·물고기·크레이트)은 `../상세/낚시_가이드.md` 참고. 여기선 **효율을 높이는 소비형 미끼·줄**만 정리합니다.

| 아이템 (영문 → 의역) | 효과 |
|---|---|
| **Bait** (미끼) | 낚싯대 미끼 슬롯에 채워 낚시 효율 보조 (우클릭으로 장전) |
| **Lucky Bait** (행운의 미끼) | 사용 시 **낚시 행운(fishing luck)** 증가 → 희귀 어획·크레이트 확률 ↑ |
| **Magnetic Bait** (자석 미끼) | 주변 물고기를 끌어당겨 입질 속도 향상 |
| **Golden Line** (황금 낚싯줄) | 장착 시 **낚시 행운 증가** (`+1 luck` 보너스) |
| **Braided / Reinforced Line** (꼰/강화 낚싯줄) | 낚싯줄 내구·성능 보조 |
| **Iron / Lavaproof / Hook of the Depths** (철/용암내성/심연 낚싯바늘) | 바늘 등급. 용암 낚시·심해 어획 가능 |

> 미끼 툴팁의 `crate chance`(크레이트 확률), `fishing luck`(낚시 행운), `fishing speed`(낚시 속도)가 곧 생산성 지표입니다.

### 3-3. 자석류 (참고)

- **자석 / 자화된 블록** (`quark:magnet`): 들고 쓰는 아이템이 아니라 **레드스톤 블록**이라 본 위키 범위 밖입니다. 아이템 자동 흡입은 여행자의 배낭 자석 업그레이드(7번 위키) 또는 낚시 자석 미끼(위 3-2)로 대체하세요.

## 4. 유지·회복 아이템 (음식·물약)

식량 보급 끊김과 자연 회복 단절을 막아 탐험·전투 지속력을 높입니다.

### 4-1. 무한 식량 — 무한한 햄 (Relics)

| 능력 | 효과 |
|---|---|
| **재생 (Autophagy)** | 사용 시 내부의 재생 가능한 고기를 소모해 **허기를 채움** (포화도 스탯 보유) |
| **연금술 (Infusion)** | 인벤토리에서 햄에 포션을 사용해 효과를 "기억"시키면, 이후 «재생»으로 먹을 때 **해당 포션 효과**가 부여됨 (충전당 지속시간 적용) |

> 사실상 **소모되지 않는 식량 + 버프 디스펜서**. 장거리 탐험·광질 상비 1순위.

### 4-2. 효율 음식 (Farmer's Delight) — 핵심 효과만

> 어떤 요리에 어떤 효과가 붙는지, 조합법 상세는 **`../상세/요리사_직업_가이드.md`** 참고.

| 효과 (영문) | 효과 설명 |
|---|---|
| **편안함 (Comfort)** | 허기 수치와 **무관하게** 자연 체력 회복 발동 → 배가 덜 차도 회복됨 |
| **배부름 (Nourishment)** | 체력 회복으로 포화를 쓸 때를 제외하면 **허기가 줄지 않음** → 식량 보급 빈도 ↓ |

| 음식 | 핵심 용도 |
|---|---|
| **사과주** (`apple_cider`) | 음용형 버프 음료 (요리사 가이드 참고) |
| **발광 열매 커스터드** (`glow_berry_custard`) | 고급 디저트, 포만·버프형 |
| **핫 코코아** (`hot_cocoa`) | **해로운 효과 1개 제거** (디버프 정화) |
| **과일 샐러드** (`fruit_salad`) | 가벼운 회복형 식사 |

> 🔑 **편안함 + 배부름**이 핵심: 전투·탐험 중 "허기 관리 없이 회복"이 가능해져, 사실상 자연 회복을 상시 켜 둡니다.

### 4-3. 디버프 정화 소모품

| 아이템 | 효과 |
|---|---|
| **핫 코코아** | 해로운 효과 1개 제거 |
| **우유병 (Milk Bottle)** | 효과 1개 제거 (병 단위라 우유 양동이보다 효율적) |

## 5. 편의 유물 (Relics)

전투보다 **이동·환경 극복·반복 작업 단축**에 쓰이는 유물. 모두 **던전/구조물 전리품 상자**에서 획득하며, 장신구(Curios) 슬롯에 장착합니다.

| 유물 | 능력 | 편의 효과 |
|---|---|---|
| **아쿠아 워커** (`aqua_walker`) | 습기 저항 | 일정 시간 **물 위를 걸음** (제한시간 후 가라앉음) |
| **마그마 워커** (`magma_walker`) | 불타는 발자국 / 내열성 | 일정 시간 **용암 위를 걸음**, 뜨거운 블록 피해 억제 |
| **양서류 부츠** (`amphibian_boot`) | 지느러미 / 아가미 | 수영 속도 ↑, 물속에서 **산소 소모 없음**(확률) |
| **아이스 스케이트** (`ice_skates`) | 스케이팅 | 얼음 위 이동 속도 대폭 ↑ |
| **롤러 스케이트** (`roller_skates`) | 가속 | 모든 블록이 미끄러워지며 **지속 가속 이동** |
| **겉날개 부스터** (`elytra_booster`) | 가속 | 연료를 채워 두고 비행 중 **전방 돌진** 부스트 |
| **블레이징 플라스크** (`blazing_flask`) | 영원한 불 | 불을 중심으로 **제한 비행 구역** 생성(크기는 불 양 비례) |
| **말 피리** (`horse_flute`) | 휴대용 마구간 | 말을 피리 안에 **수납/소환**, 보관 중 자동 치유. 16블록 초과 시 자동 귀환 |
| **포자 주머니** (`spore_sack`) | 독 합성 / 분해 / 독 방출 | 피격 시 포자를 뿌려 **자동 방어**(중독·둔화·치유차단), 다단계 활용 |
| **드라운드 벨트** (`drowned_belt`) | 부하 용량 | **장신구 슬롯 +α**, 수중 전투·급류(삼지창) 강화 |
| **헌터 벨트** (`hunter_belt`) | 부하 용량 | **장신구 슬롯 +α**, 펫(길들인 동물) 피해 강화 |
| **가죽 벨트** (`leather_belt`) | 부하 용량 | **부적(Charm) 슬롯 +α** — 유물 운용 확장의 기본 |

> 💡 **벨트류(가죽/드라운드/헌터)** 는 장신구·부적 슬롯을 늘려 줘, 다른 편의 유물을 더 많이 장착하게 해 주는 "확장 카드"입니다. 유물 빌드의 토대.

## 6. 시간·경험 절약 아이템

인챈트 시행착오와 경험치 낭비를 줄여 주는 소모품 (대부분 Apotheosis).

### 6-1. 인챈트 효율 소모품 (Apotheosis)

| 아이템 | 효과 |
|---|---|
| **폐기의 책** (`scrap_tome`) | 인챈트된 아이템을 갈아 **인챈트를 책으로 회수**(폐기). 인챈트 재활용 |
| **최상급 폐기의 책** (`improved_scrap_tome`) | 폐기의 책 상위판. 더 높은 회수 효율 |
| **추출의 책** (`extraction_tome`) | 아이템에서 인챈트/속성을 추출 |
| **프리즈마틱 거미줄** (`prismatic_web`) | 인챈트 부여 결과를 다시 굴리는(리롤) 보조 소재 |
| **부위별 인챈트 책** (`weapon_tome` / `pickaxe_tome` / `bow_tome` / `fishing_tome` / `helmet_tome` 등) | 특정 장비 슬롯 대상의 인챈트 작업용 책 |
| **소켓의 인장** (`sigil_of_socketing`) | 장비에 **보석 소켓 추가** (장비 강화 슬롯 확보) |
| **최고급 소켓의 인장** (`superior_sigil_of_socketing`) | 소켓 인장 상위판 |
| **이름 없는 약병** (`vial_of_unnaming`) / **추출 약병** (`vial_of_extraction`) | 보석/속성 이름·부여 제거·추출용 소모 약병 |

> 🔑 **폐기의 책**은 못 쓰는 인챈트 장비를 책으로 되살려 인챈트 자원을 아끼는 핵심. 인챈트 도서관(7번 인프라)과 함께 쓰면 시행착오 비용이 크게 줄어듭니다.

### 6-2. 경험치 절약

| 아이템 | 효과 |
|---|---|
| **유물 경험치 병** (`relic_experience_bottle`) | 모아 둔 경험치를 **유물 레벨업에 직접 투입** → 유물 강화 시간 단축 |
| **전송 비용 관리** | 전송석 계열 전송은 XP를 소모하므로, **연결 주문서/전송판**으로 짧은 거리 위주로 동선을 짜면 XP 절약 (2-1 참고) |

> ⚠️ 무한한 햄·엔더 핸드·매직 미러 등 유물은 사용할 때마다 자체 경험치가 쌓입니다. 평소 이동·식사에 유물을 쓰는 것만으로도 유물이 알아서 강화됩니다.
