# 📜 의뢰·교역 위키

> **다루는 범위**: 반복 수행으로 돈·경험치·자원을 버는 **의뢰판(현상금판) + 특수 상인 교역** 루프.
> 거래소(Trading Post) 제작 등 생산성 측면은 `7_생산성툴_위키.md`, 콜로니 경제 전반은 `모드팩_플레이_가이드.md` 를 참고하세요.
> 모든 항목은 각 모드 jar의 실제 언어 파일(`en_us.json` / 일부 `ko_kr.json`)에서 추출했습니다. 메커니즘 수치가 파일에 없는 부분은 "확인 필요"로 표기합니다.

---

## 1. 개요 — 반복 수익 루프란?

초반 자금·경험치·자원이 마른 상황에서 **반복적으로 돌릴 수 있는 수입원**을 모아 정리합니다. 크게 두 갈래입니다.

| 갈래 | 무엇 | 핵심 모드 |
|---|---|---|
| 📋 의뢰 루프 | 게시판이 던지는 과제를 수행 → 보상 회수 | Bountiful (의뢰 게시판) |
| 🤝 교역 루프 | 특수 상인·신규 직업 주민과 거래 | Goblin Traders, Nether Villager Trader, Villagers Plus |

> 의뢰는 "내가 가진/만들 수 있는 물건을 보상으로 환전"하는 수단이고, 교역은 "특정 자원을 다른 자원으로 바꾸는" 수단입니다. 두 루프를 병행하면 초반 자금 회전이 크게 빨라집니다.

---

## 2. 📋 의뢰 게시판 (Bountiful)

**모드**: Bountiful · **블록**: `의뢰 게시판` (Bounty Board)
반복 의뢰를 받아 요구 물품/처치/경험치를 채우고, 보상으로 환전하는 시스템입니다.

### 2-1. 기본 흐름

| 단계 | 행동 | 파일 근거 |
|---|---|---|
| ① 의뢰 받기 | 의뢰 게시판에서 의뢰(Bounty)를 수령 | 업적 "고용된 몸 / Work for Hire": *의뢰 게시판에서 의뢰를 받으세요* |
| ② 수행 | 의뢰서에 적힌 **요구사항(Required)** 충족 (납품·처치·경험치 등) | 툴팁 `요구사항`, `보상`, `완료까지 남은 시간` |
| ③ 완수 | **의뢰 게시판에 우클릭**하여 의뢰 완수 → 보상 회수 | `의뢰 게시판에 우클릭 해서 의뢰를 완수하세요` |

> ⏳ 의뢰에는 **제한 시간(Time To Complete / 완료까지 남은 시간)** 이 있습니다. 시간이 지나면 `This bounty has expired.`(의뢰 만료) 상태가 되어 보상을 받을 수 없습니다.

### 2-2. 의뢰 종류 (요구 유형)

언어 파일에서 확인되는 의뢰 목표 유형:

| 유형 | 설명 | 예시(파일) |
|---|---|---|
| 🧺 아이템 납품 | 지정 아이템을 게시판에 제출. **와일드카드** 허용 항목 다수 | Any Wool(아무 양털), Any Planks(아무 판자), Any Log(아무 원목), Any Bed(아무 침대), Any Boat(아무 보트), Any Candle(아무 양초), Any Banner(아무 현수막) |
| ⚔️ 처치 (Kills) | 특정 몹을 일정 수 처치 | `bountiful.bounty.type.entity.kills` = 처치 |
| ✨ 경험치 | 경험치 **레벨(Experience Levels)** 또는 **포인트(Experience Points)** 제출 *(미번역)* | `experience.levels`, `experience.points` |
| 🛠️ 행동 과제 | 특정 행위 수행 | Fish up Enchanted Books(마법책 낚기), Brew a basic Swiftness Potion(기본 신속 물약 양조), Cure a Zombie Villager(좀비 주민 치료) |

### 2-3. 포고문(Decree) — 의뢰 분야 지정

의뢰 게시판이 어떤 **분야**의 의뢰를 내줄지는 **포고문(Decree / 포고문)** 으로 결정됩니다.

- 포고문 아이템을 입수(`마을 포고꾼` 업적) → **의뢰 게시판에 붙이면**(`왕의 칙령` 업적) 해당 분야 의뢰가 활성화됩니다.
- 안내 문구: *이것을 의뢰 게시판에 붙이십시오! (`Bring this to a Bounty Board!`)*

**포고문 분야 일람** (한글 번역 기준, 일부 미번역은 영문 병기):

| 분야 | 한글 | 분야 | 한글 |
|---|---|---|---|
| Armorsmithing | 갑옷 제조 | Librarian | 사서 |
| Butchering | 도축 | Cartography | 지도 제작 |
| Chef's Supplies | 요리사 납품 | Masonry | 석공 *(미번역: Masonry)* |
| Cleric's Supplies | 성직자 납품 | Shepherding | 양치기 |
| Farming | 농사 | Tinkering | 수선 *(미번역: Tinkering)* |
| Fishing | 낚시 | Toolsmithing | 도구 제조 |
| Fletching | 화살 제조 | Inventing | 발명 *(미번역: Inventing)* |
| Mob Hunting | 현상금 사냥 | Leatherworking | 가죽 세공 |

> 위 분야 명칭은 바닐라 주민 직업군과 대응됩니다. 자신이 잘 모으는 자원(예: 양털→양치기, 작물→농사)에 맞는 포고문을 붙이면 회전이 빠릅니다.

### 2-4. 등급(Rarity)과 통계

의뢰는 **희귀도 등급**이 있으며, 등급별 완수 횟수가 별도로 집계됩니다.

| 등급 (en) | 한글 |
|---|---|
| Common | 흔한 |
| Uncommon | 드문 |
| Rare | 희귀한 |
| Epic | 놀라운 |

집계 통계: `받은 의뢰 수`, `완수한 의뢰 수(총합)`, 그리고 등급별 `완수한 의뢰 수(흔한/드문/희귀한/놀라운)`.

> 💡 등급이 높을수록 요구가 까다롭고 보상도 큰 구조로 추정됩니다(구체 보상 테이블은 파일에 없음 → **확인 필요**). 게시판 평판/숙련 같은 별도 등급 상승 시스템은 언어 파일에 없습니다.

### 2-5. 시간 관련 업적 (참고)

빠른 완수를 노릴 때 따라오는 업적:

| 업적 | 조건 |
|---|---|
| 총알배송 (Rush Order Delivery) | 의뢰를 **1분 안에** 완수 |
| 더 이상 미룰 순 없다 (Master Procrastinator) | 남은 시간 **5초 이내**에 완수 |

## 3. 🤝 특수 상인

### 3-1. 고블린 상인 (Goblin Traders)

**모드**: Goblin Traders · 떠돌이 상인 계열의 특수 교역 NPC입니다.

| 엔티티 | 한글(의역) | 등장 추정 |
|---|---|---|
| Goblin Trader | 고블린 상인 | 지상(오버월드) |
| Vein Goblin Trader | 광맥 고블린 상인 | 지하 깊은 곳(광맥대) — 명칭상 |

- 통계 `Traded with Goblins`(고블린과 거래함)로 거래 횟수가 집계됩니다.
- 스폰 알 아이템 존재: `Goblin Trader Spawn Egg`, `Vein Goblin Trader Spawn Egg`.
- 소리/자막: `Annoyed Grunt`(짜증 섞인 으르렁), `Idle Grunt`(평상시 으르렁) — 근처에 있으면 소리로 위치 파악 가능.
- 언어 파일에 **희귀 물약류 거래**가 암시됩니다: 신속(Haste), 흡수(Absorption), 돌고래의 우아함(Dolphins Grace) — 일반/투척/잔류 형태 전부 등장. 바닐라로는 못 만드는 효과라 **교역 가치 높음**.

> 📌 구체 교역 품목·가격 테이블은 언어 파일에 없습니다(**확인 필요**). 다만 위 물약 항목으로 보아 "특수 효과 물약을 사는 창구"로 활용 가치가 큽니다.

### 3-2. 네더 고블린 상인

위 **Vein Goblin Trader(광맥 고블린 상인)** 가 깊은 지하/광맥대에서 등장하는 변종입니다. 네더 전용 별도 엔티티는 언어 파일상 확인되지 않으며, 지하 깊이 기반으로 추정됩니다(**확인 필요**).

### 3-3. 네더 주민 상인 (Nether Villager Trader)

**모드**: Nether Villager Trader
주민을 **네더 상인(Nether Trader)** 직업으로 전직시켜, 지옥 테마 장비를 거래로 얻는 시스템입니다.

| 요소 | 내용 |
|---|---|
| 전직 블록 | `Nether Trader Profession Block` — *주민을 네더 상인으로 바꿉니다(Turns a Villager into a Nether Trader)* |
| 직업 엔티티 | `Nether Trader`(네더 상인) |
| 통화 추정 | `Hell Ingot`(지옥 주괴) — *지옥 단조 장비를 수리. 지옥검으로 제작 가능* |

**거래/제작으로 얻는 지옥(Hell) 장비** (효과는 파일 description 기준):

| 아이템 | 효과 |
|---|---|
| Hell Sword (지옥검) | 적을 불태움 |
| Hell Axe (지옥도끼) | 명중 시 작은 폭발 발생 |
| Hell Sickle / Nether Sickle (지옥낫) | 명중 시 위더 효과 부여, 우클릭으로 전방 대시 |
| Hell Armor (헬멧/흉갑/레깅스/부츠) | **세트 보너스: 화염 저항(Fire Resistance)** |
| Hell Potion (지옥 물약) | 가스트 보호(Ghast protection) 부여 |
| Hell Ingot (지옥 주괴) | 지옥 단조 장비 수리 재료, 지옥검 제작 재료 |

전용 효과: `Ghast protection`(가스트 보호) — 가스트 화염구 피해 경감 추정.

> 📌 정확한 거래 슬롯/가격은 파일에 없음 → **확인 필요**. 네더 진출 시 화염 저항 풀세트를 거래로 확보할 수 있어 가성비가 높습니다.

## 4. 🧑‍🌾 신규 주민 직업 (Villagers Plus)

**모드**: Villagers Plus · 바닐라에 없는 **신규 주민 직업 4종**과 각 직업의 **직업 블록**을 추가합니다. (각 직업의 구체 거래 품목·가격은 언어 파일에 없음 → **확인 필요**)

| 직업 (en) | 한글(의역) | 직업 블록 | 테마 |
|---|---|---|---|
| Alchemist | 연금술사 | Alchemist Table(연금술 작업대) | 물약/연금 관련 |
| Occultist | 강령술사 | Enchanted Basin(마법 분지) | 마법부여/주술 관련 |
| Oceanographer | 해양학자 | Aquarium(수족관) | 바다 자원 관련 |
| Horticulturist | 원예가 | Flower Tub(화분, 목재별 변형 다수) | 꽃·식물 관련 |

> 🌸 원예가의 화분(Flower Tub)은 참나무·자작나무·정글·아카시아·짙은참나무·맹그로브·진홍빛·뒤틀린·벚나무·대나무 등 **목재별 변형 블록**이 존재합니다. 거점 미관 + 직업 부여를 겸합니다.

이들 직업은 바닐라 거래 메커니즘(에메랄드 매개)을 따르므로, 에메랄드 수급 루프(농사·낚시 의뢰 등)와 잘 맞물립니다.

## 5. 🏪 거래소(Trading Post) 연계

**모드**: Trading Post · 주변 주민들의 거래를 **한 블록에서 통합 조회·거래**하는 장치입니다.

- 검색창(`Search...`)으로 원하는 거래를 빠르게 찾습니다.
- 근처에 거래 가능한 주민이 없으면 `Couldn't find any available trader nearby`(근처 거래 가능한 상인 없음), 거래 중 상인이 사라지면 `The trader is no longer available.` 안내가 뜹니다.

> 🔧 거래소 **제작법·배치·상세 운용**은 `7_생산성툴_위키.md` 를 참고하세요. 본 위키에서는 "여러 주민(특히 Villagers Plus 신규 직업 포함)을 한곳에 모아 의뢰·교역 루프를 가속하는 허브" 역할로만 다룹니다.

## 6. 💰 추천 수익 루프 (초반 돈·경험치 벌이)

파일에서 확인된 사실을 기반으로 한 실전 루프 제안입니다.

### 루프 A — 의뢰 환전 (자원 → 보상)

1. 거점에 **의뢰 게시판** 설치 → 내가 잘 모으는 자원에 맞는 **포고문** 부착(예: 양털 다발 → 양치기, 작물 → 농사).
2. **와일드카드 납품 의뢰**(아무 양털/판자/원목 등)를 우선 수령 — 잡템을 보상으로 환전.
3. 제한 시간 내 게시판에 우클릭으로 완수, 등급(흔한→놀라운) 올려가며 반복.

### 루프 B — 교역 환전 (특수 상인)

1. 지하 채굴 중 **고블린 상인** 조우 → 신속/흡수/돌고래의 우아함 등 **특수 물약** 확보.
2. 네더 진출 전 주민을 **네더 상인**으로 전직 → **화염 저항 세트(Hell Armor)** 와 가스트 보호 물약 확보.
3. **거래소**로 일반 주민 + Villagers Plus 신규 직업(연금술사·해양학자 등) 거래를 통합 운용.

### 루프 C — 경험치 회수

- 경험치 유형 의뢰(Experience Levels/Points)는 남는 경험치를 보상으로 바꾸는 창구가 될 수 있습니다(보상 내역 **확인 필요**).
- 처치(Kills) 의뢰 + 현상금 사냥(Mob Hunting) 포고문 조합으로 전투를 수익화.
